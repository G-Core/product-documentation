## Activate Streaming

Streaming is available in the reseller admin portal only if the reseller has the **Streaming** product enabled.

To activate Streaming for a customer:

1. Open the [Admin Portal](https://admin.gcore.top/).
2. Go to the **Streaming** section.
3. Create a new customer account or open an existing one.
4. Configure the required **Streaming limits**.

By default, Streaming services are disabled for new customers until the required limits are configured.

## Streaming Admin Portal (IAM)

The **Streaming Admin Portal (IAM)** provides an interface for managing customers who use the Gcore Streaming platform.

### Interface overview

The portal displays a list of all customers using Streaming, including customers on **trial** and **free** tariff plans.

## Manage trial limits

When a new Streaming account is created, all streaming limits are set to `0` by default, regardless of the account status.

For customers requesting access to live streaming for the first time, the following limits must be configured before the service can be used:

- **Streams count** — maximum number of simultaneous live streams.
- **Restreams count** — number of simultaneous restream destinations.
- **Video storage minutes** — amount of minutes available for storing video recordings.
- **Encoding video minutes** — available minutes for video transcoding.
- **VOD storage (GB)** — available storage capacity for video-on-demand content.

These limits can be configured in the **Edit account** settings in the Streaming Admin Portal.

### Trial configuration

For trial accounts, the limits are typically configured with restricted values. Example trial limits may include:

- **Streams count:** 1  
- **Restreams count:** 1  
- **Video storage minutes:** 30  
- **Encoding video minutes:** 30  
- **VOD storage:** 1000 GB  

Resellers can adjust these values depending on the trial conditions agreed with the customer.

## Free features

Some Streaming features can be enabled for customers as free features in the account settings.

Available free features may include:

- **video_editor** — allows editing uploaded videos directly in the platform.
- **browsercast** — enables browser-based live streaming.
- **original_recording** — allows storing the original stream recording.
- **html_overlays** — enables HTML overlays for live streams.
- **closest_transcoder_routing** — routes streams to the nearest transcoder for improved performance.
- **audit_log** — provides access to activity logs for Streaming operations.

Free features can be enabled or disabled individually for each customer account.

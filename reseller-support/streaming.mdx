## Activate Streaming

Streaming is available in the reseller admin portal only if the reseller has the **Streaming** product enabled.

## General Information

The **Streaming** platform can be managed through:

- **Admin Portal (AP)**
- **VP-Admin**
- **Customer Portal (CP)**

There are two admin portals for Streaming:

- **Streaming Admin Portal (IAM)**
- **VP Admin Portal**

## Streaming Admin Portal (IAM)

The **Streaming Admin Portal (IAM)** provides an interface for managing customers who use the Gcore Streaming platform.

### Interface overview

The portal displays a list of all customers using Streaming, including customers on **trial** and **free** tariff plans.

### Streaming limits

By default, the **stream limit** is set to **0** for new customers, regardless of their account status.

For customers requesting access to **live streaming** for the first time, Streaming, Restream, and Storage limits must be configured before the service can be used.

## VP Admin Portal

The **VP Admin Portal** is an additional tool used by the Support and Streaming teams to analyze and manage streams, videos, and related CDN resources for customers using the Gcore Streaming platform. The portal also provides access to transcoding logs and other diagnostic information useful for troubleshooting and analysis.

The VP Admin Portal is available at the internal VP Admin interface. Access credentials are stored together with other service passwords.

## Customer Portal (Streaming Platform)

The **Customer Portal** provides customers with access to Streaming platform modules and features.

### Available modules and features

- **Video hosting** — upload videos, create folders, update video metadata, configure players and ads, assign tags, and view statistics.
- **Live streaming** — create live streams, restreams, and multicamera streams, configure streaming settings, view monitoring data and statistics.
- **Players** — create and configure video players.
- **Ads** — create ad templates and activate specific ads.
- **Statistics** — view video, live streaming, and billing statistics.

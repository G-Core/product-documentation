BGP enables the announcement of IP addresses using Gcore equipment. The service is available only on Dedicated Servers.

Two services are available under **BGP and Subnetwork announce**:

- **BGP** — announces IP addresses from a custom AS number via Gcore infrastructure. Price: 30 EUR/month.
- **Subnetwork announce via route object (our ASN)** — announces a subnet via Gcore's ASN using a route object. Price: 70 EUR/month. Available only in Kazakhstan; contact technical support to set it up.

To activate BGP or Subnetwork announce:

1. Log in to the [Control&nbsp;Panel](https://hosting.gcore.com/billmgr).
2. In the left sidebar, navigate to **Products/Services** > **BGP and Subnetwork announce**.
3. Click **Order** next to the service to activate.
4. Open a support ticket to request configuration.

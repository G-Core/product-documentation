Link aggregation (LACP bonding) combines multiple Ethernet interfaces into a single logical channel. The feature distributes server load and increases fault tolerance. Gcore uses the standard LACP protocol (IEEE 802.3ad) for channel bonding. The service is available on Dedicated Servers only and is paid — prices are listed on the [LACP&nbsp;tab](https://gcore.com/hosting/miscellaneous).

To activate link aggregation, order the **Port Channel** service from the [Control&nbsp;Panel](https://hosting.gcore.com/billmgr) and open a support ticket — Gcore configures the network-side LACP after receiving the request.

To order the Port Channel service:

1. Log in to the [Control&nbsp;Panel](https://hosting.gcore.com/billmgr).
2. In the left sidebar, navigate to **Products/Services** > **Port Channel**.

<Frame><img src="/images/docs/hosting/other-services/link-aggreagtion/activate-link-aggregation/activate-link-aggregation-image1.png" alt="Port Channel section in the hosting Control Panel with the Order button" width="70%" /></Frame>

3. Click **Order** next to the Port Channel service.

<Frame><img src="/images/docs/hosting/other-services/link-aggreagtion/activate-link-aggregation/activate-link-aggregation-image2.png" alt="Port Channel order pricelist showing 1.00 EUR per month with Order button" width="70%" /></Frame>

4. Open a support ticket and specify the server and the required configuration. Gcore configures the link aggregation after the request is received.

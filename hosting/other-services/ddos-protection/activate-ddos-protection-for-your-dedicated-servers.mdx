We provide DDoS Protection for dedicated (physical) servers, including servers rented from us and from third parties. You can also rent a new protected server from us.

1. To order protection for a rented server, [send a request through technical support](https://gcore.com/docs/hosting/contact-our-technical-support). Please note: We will change the server's main IP address after we set up the protection. Specify the following in your request:
   - Server ID
   - OSI layers to protect (L3-L4 or L3-L7)
   - Bandwidth
   - IP whitelist and blacklist (optional, you can send them later)
2. To order protection for a third-party server, fill out an application at the bottom of the [DDoS Protection](https://gcore.com/ddos-protection-for-servers) page on our website. Specify the following in the application:
   - Your contact details
   - Equipment location
   - Equipment subnet
   - IP whitelist and blacklist (optional, you can send them later)
3. To rent a new server with DDoS Protection, select a server configuration in the [Dedicated Servers](https://gcore.com/hosting/dedicated) section of our website, then fill out an application at the bottom of the page. Specify the following in the application:
   - Server configuration
   - OSI layers to protect (L3-L4 or L3-L7)
   - Bandwidth
   - IP whitelist and blacklist (optional, you can send them later)

## Deactivate service

To deactivate the protection,[ write to our technical support](https://gcore.com/docs/hosting/contact-our-technical-support). Please note: the server's main IP address will be changed after deactivation.

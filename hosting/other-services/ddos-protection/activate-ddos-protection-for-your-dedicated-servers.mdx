DDoS Protection is available for dedicated (physical) servers — both rented from Gcore and third-party. A new server can also be ordered with protection already included.

<Warning>
The main IP address of the server will change after protection is set up or deactivated.
</Warning>

## Rented Gcore server

To activate protection for a rented server, contact [technical&nbsp;support](/hosting/contact-our-technical-support) and include the following details:

* Server ID
* OSI layers to be protected (L3-L4 or L3-L7)
* Bandwidth
* Allowlist and blocklist of IP addresses (optional — can be provided later)

## Third-party server

To activate protection for a third-party server, contact [technical&nbsp;support](/hosting/contact-our-technical-support) and include the following details:

* Contact details
* Location of the equipment
* Subnet of the equipment
* Allowlist and blocklist of IP addresses (optional — can be provided later)

## New server order

To order a new server with DDoS Protection included, contact [technical&nbsp;support](/hosting/contact-our-technical-support) and include the following details:

* Server configuration
* OSI layers to be protected (L3-L4 or L3-L7)
* Bandwidth
* Allowlist and blocklist of IP addresses (optional — can be provided later)

## Deactivate service

To deactivate the protection, contact [technical&nbsp;support](/hosting/contact-our-technical-support).

Multiple people often need access to the same hosting account — a site administrator, a billing contact, or a developer. Adding a sub-user gives each person separate login credentials instead of sharing a single password.

To add a new user:

1. In the Control Panel, navigate to **Tools** > **Add new user**.

<Frame>![Add new user form with email, password, login, role, and avatar fields](/images/docs/hosting/account-management/users/add-a-new-user-to-your-account/add-a-new-user-to-your-account-image1.png)</Frame>

2. Fill in the required fields:
   - **Email** — the login email address for the new user.
   - **Password** and **Confirmation** — the initial password. The user can change it after logging in.
3. (Optional) Fill in additional fields as needed:
   - **Login** — an alternative username.
   - **Full name** — the user's display name.
   - **Phone number** — a contact number.
   - **Full access** — grants the user full account permissions. When enabled, **User roles** has no effect.
   - **User roles** — assigns a specific role with limited permissions. Use this instead of **Full access** to restrict what the user can do.
   - **Avatar** — uploads a profile photo.
4. Click **Save**.

The new user appears under **Client** > **Users**. This section becomes available after the first sub-user is added.

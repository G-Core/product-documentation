The hosting portal logs all access events for the account, including logins, logouts, and session resets.

To view the authorization history:

1. In the left sidebar, navigate to **Statistics**.
2. Click **Access log**.

<Frame>
<img src="/images/docs/hosting/account-management/view-the-authorization-history-of-your-account/view-the-authorization-history-of-your-account-image1.png" alt="Access log table showing authorization events with time, user, IP, and action columns" width="70%" />
</Frame>

The following data is displayed for each event:

- **Time** — date and time of the event
- **User** — email address and account ID of the user
- **Remote IP address** — IP address of the device used to access the account
- **Remote port** — port used for the connection
- **Action** — event type: Authorization, Logout, or Reset session
- **Authorization method** — login method used for the event; "With a login" appears for standard logins

ISO images for virtual servers are managed in VMmanager 6. Log in to the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), navigate to **Products/Services** → **Virtual private servers**, click the three-dot menu on the server row, and select **ISO images**.

The [ISO&nbsp;installation](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image) article covers the full connection process.

## Local file upload fails

If the ISO image upload from a local file fails, check the following:

- The file extension must be lowercase: `.iso`, not `.ISO`.
- The filename must contain only letters and numbers.
- The file size must not exceed 8.6 GB.

## Image URL connection fails

If connecting an ISO image via URL fails, check the following:

- The URL must be a direct link to the ISO file, accessible over HTTP or HTTPS.
- The server hosting the file must allow external downloads.

If the error persists, [contact&nbsp;support](/hosting/contact-our-technical-support).

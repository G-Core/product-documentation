A Virtual Server is a physical machine partitioned into isolated virtual environments using KVM (Kernel-based Virtual Machine) virtualization. Each server runs its own operating system kernel with guaranteed RAM, disk, and vCPU allocation, and a private network interface — no resources are shared with other servers on the same host.

Virtual Servers are available in two plan categories:

- **KVM-SSD** — balanced configurations for general-purpose workloads: web servers, databases, VPN, and game servers.
- **KVM-HIGHCPU-SSD** — configurations with a higher vCPU-to-RAM ratio for CPU-intensive tasks: video transcoding, scientific computing, and high-concurrency applications.

Both categories use SSD storage and include SSH access and a VNC console. The [plan&nbsp;comparison](/hosting/virtual-servers/before-purchase/choose-a-virtual-server-for-your-needs) covers all configurations in detail.

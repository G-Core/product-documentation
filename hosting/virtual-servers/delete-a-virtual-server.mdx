Deleting a virtual server permanently removes it from the account, erases all data, and stops billing.

<Warning>
Deletion is irreversible. Back up any important files before proceeding.
</Warning>

## Delete a server

1. In the Gcore Hosting [control&nbsp;panel](https://hosting.gcore.com/billmgr), navigate to **Virtual private servers** in the left-side menu.
2. Select the checkbox next to the server and click **Delete** in the toolbar.

<Frame>
  ![Virtual private servers list with a server selected and the Delete button highlighted in the toolbar](/images/docs/hosting/virtual-servers/delete-a-virtual-server/delete-a-virtual-server-image1.png)
</Frame>

3. In the confirmation dialog, click **Ok**.

<Frame>
  ![Confirmation dialog asking to confirm server deletion, with Ok and Cancel buttons](/images/docs/hosting/virtual-servers/delete-a-virtual-server/delete-a-virtual-server-image2.png)
</Frame>

The server is removed from the list, all data is erased, and billing stops.

## Minimum service term

Virtual servers have a one-month minimum service term. If a deletion attempt is blocked, a warning icon appears on the server row. Hover over the icon to see how many days remain before deletion is allowed.

<Frame>
  ![Warning icon tooltip showing the remaining days before deletion is allowed](/images/docs/hosting/virtual-servers/delete-a-virtual-server/delete-a-virtual-server-image3.png)
</Frame>

To resolve other deletion errors, contact [support](/hosting/contact-our-technical-support).

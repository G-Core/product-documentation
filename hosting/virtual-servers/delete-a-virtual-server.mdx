Deleting a virtual server permanently erases all data and stops billing. The server disappears from the list immediately.

## Deleting a server

In the [Control Panel](https://hosting.gcore.com/billmgr), navigate to **Products/Services** → **Virtual private servers**.

<Steps>
  <Step title="Select the server">
    Click the server row to select it. The row highlights and the toolbar buttons become active.

    <Frame>![Virtual private servers list with a server selected and Delete button active in the toolbar](/images/docs/hosting/virtual-servers/delete-a-virtual-server/delete-a-virtual-server-image1.png)</Frame>
  </Step>
  <Step title="Click Delete and confirm">
    Click **Delete** in the toolbar. A confirmation dialog titled **ATTENTION!** appears, showing the server's plan name, ID, IP address, and domain name.

    <Frame>![Deletion confirmation dialog with Ok and Cancel buttons](/images/docs/hosting/virtual-servers/delete-a-virtual-server/delete-a-virtual-server-image2.png)</Frame>

    Click **Ok** to confirm. The server disappears from the list immediately. Click **Cancel** to abort.
  </Step>
</Steps>

<Warning>Deletion is permanent — the server and all its data cannot be restored.</Warning>

## Deletion error: minimum service term

If deletion is blocked, the server was likely purchased less than a month ago. The minimum service term is one month and cannot be waived. To delete the server, wait until the one-month period has elapsed and try again.

To see the error details, hover over the exclamation mark icon on the server row.


<Frame>![Virtual private servers list showing a deletion error icon on the server row](/images/docs/hosting/virtual-servers/delete-a-virtual-server/mceclip11.png)</Frame>



For other errors, the [support team](/hosting/contact-our-technical-support) can help.

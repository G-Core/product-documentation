Virtual Servers are available in several plan categories that differ by disk type and CPU allocation. Comparing the categories and their specifications helps identify the right plan for a given workload.

## Available plans

In the [Virtual server (VDS) catalog](https://hosting.gcore.com/billmgr?startform=v2.vds.order.pricelist), select a data center to filter the list and view available plans with their specifications.

<Frame>![VDS plan list filtered to Luxembourg, DC 2 in BillMgr](/images/docs/hosting/virtual-servers/before-purchase/choose-a-virtual-server-for-your-needs/choose-a-virtual-server-for-your-needs-image1.png)</Frame>

The [Gcore website](https://gcore.com/hosting/vds) provides a full plan list with filters for data center, CPU cores, RAM, disk space, OS, and price. Apply the filters to narrow the list to matching plans.

<Frame>![Virtual Server configurator on the Gcore website with filter options](/images/docs/hosting/virtual-servers/before-purchase/choose-a-virtual-server-for-your-needs/choose-a-virtual-server-for-your-needs-image2.png)</Frame>

Two plan categories are available: **KVM-SSD** and **KVM-HIGHCPU-SSD**. A category is indicated in the name of a plan.

<Frame>![BillMgr VDS order page showing KVM-SSD plan cards for Luxembourg, DC 2](/images/docs/hosting/virtual-servers/before-purchase/choose-a-virtual-server-for-your-needs/choose-a-virtual-server-for-your-needs-image3.png)</Frame>

<Frame>![BillMgr VDS order page showing KVM-HIGHCPU-SSD plan cards for Luxembourg, DC 2](/images/docs/hosting/virtual-servers/before-purchase/choose-a-virtual-server-for-your-needs/choose-a-virtual-server-for-your-needs-image4.png)</Frame>

**KVM-SSD** plans are suitable for a wide range of tasks. All plans in this category use SSD storage.

Windows Server OS is not available on KVM-SSD-1, KVM-SSD-2, and KVM-SSD-3 plans.

**KVM-HIGHCPU-SSD** plans are suitable for CPU-intensive tasks. Compared to KVM-SSD plans of a similar size, these plans allocate more CPU cores and less RAM.

Regardless of the plan, each server has one IPv4 and one IPv6 address. Up to nine additional IPv4 and nine IPv6 addresses can be purchased for any plan. Entry-level plans support only one additional address of each type. The OS can also be replaced using a [template OS](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template) or a [custom ISO image](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image).

## Choose a plan for your needs

To deploy a VPN, Apache, NGINX, or a game server, KVM-SSD plans are suitable — these workloads do not require high CPU performance.

For CPU-intensive tasks, choose KVM-HIGHCPU-SSD servers.

If a Virtual Server does not provide sufficient performance, [order a Dedicated Server](/hosting/dedicated-servers/order-a-dedicated-server) instead.

When the required resources are uncertain, start with any plan and validate it against the actual workload. [Plan upgrades](/hosting/virtual-servers/upgrade-your-virtual-server) are available — contact technical support and specify the desired configuration.

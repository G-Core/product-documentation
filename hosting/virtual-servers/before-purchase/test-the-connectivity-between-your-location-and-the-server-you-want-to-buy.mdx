To test the connectivity between an IP address and one of our nodes, use the [Looking Glass](https://lg.gcore.lu) tool.

<Steps>
  <Step title="Select a test command">
    - **BGP** — shows the BGP routing path from the selected router to the destination IP address, including all autonomous systems (AS) along the route. After the test, click **Show BGP map** to view the AS path as a diagram.
    - **PING** — shows the round-trip time (RTT) and time to live (TTL) for the route between the selected router and the destination IP address.
    - **TRACEROUTE** — displays all router hops in the path between the selected router and the destination IP address, including transit time for each hop.
  </Step>
  <Step title="Select a region">
    Select a **Region** from the drop-down list to filter available routers by geographic area.
  </Step>
  <Step title="Select a router">
    Select the router location from the **Router** drop-down list.
  </Step>
  <Step title="Enter the IP address">
    Enter the IPv4 or IPv6 address to test.
  </Step>
  <Step title="Run the test">
    Click **Run test**.
  </Step>
</Steps>

The tool may take some time to process the request, depending on the selected location.

After the test completes, the results appear below the form. Use the action buttons to work with the results:

- **Copy to clipboard** — copies the raw result text.
- **Open results page** — opens the results at a permanent URL for sharing.
- **Show BGP map** — displays the AS path as a visual diagram (available for BGP tests only).

Select a Virtual Server plan from the [Gcore hosting catalog](https://hosting.gcore.com/billmgr), configure the order, and complete payment — the server activates automatically and is ready within up to 3 hours.

## Configure the order

After selecting a plan, the order form opens. Configure the following options:

- **Payment period** — select the billing cycle. The minimum initial period is one month.
- **Operating system** — select the OS to install on the server.
- **Domain name** — enter a domain to link to the server, or leave blank to use example.com (can be changed later).
- **Public IPv4-address** — use the slider or number input to set the number of IPv4 addresses (minimum 1). The cost summary updates in real time.
- **IPv6 network** — included at no extra cost.
- **Number of services** — order multiple identical servers at once (up to 10).

<Note>
When linking a custom domain, register it with a third-party provider before ordering — the hosting panel does not support domain registration. The domain must be second-level or higher, with a maximum of 63 characters per label and 255 characters in total.
</Note>

## Complete the purchase

The total cost shown in the summary reflects all selected options. Click **Pay** to go directly to payment, or click **To cart** to save the order to the cart for later.

## Activation

A server can remain in **Activation in progress** status for 15 minutes to 3 hours, after which it transitions to **Active** status and is ready to use.

If the server was saved to the cart, it appears in **Ordered** status until payment is completed. Verify the account balance and add funds if needed — the server then transitions to **Activation in progress** status.

If the server hasn't transitioned to **Active** status within 3 hours, contact support via chat or ticket from the Control Panel.

## Renewal

After the initial paid period ends, the server switches to daily billing automatically. The account balance is charged at a proportional rate — for a 30-day month, 1/30 of the plan cost is charged each day.

The Gcore hosting catalog lists available virtual server plans. Select a plan to open the order form, configure the server, and complete the purchase — the server activates automatically after payment.

## Order form

After selecting a plan (on the website or the shop window), the order form opens.

<Frame>![VDS order form with OS selection, domain name field, and IPv4 quantity slider](/images/docs/hosting/virtual-servers/order-a-virtual-server/order-a-virtual-server-image1.png)</Frame>

The order form shows pricing and the following configuration options:

- **Payment period** — select the billing cycle. The minimum initial period is one month.
- **Operating system** — select the OS to install on the server.
- **Domain name** — enter a domain to link to the server, or leave blank to use example.com (can be changed later). Register the domain with a third-party provider before ordering when linking a custom domain — the hosting panel does not provide domain registration. The domain must be at least second-level, with no more than 63 characters per level and 255 characters total.
- **Public IPv4-address** — use the slider or number input to set the number of IPv4 addresses (minimum 1). The cost summary updates in real time as the quantity changes.
- **IPv6 network** — included at no extra cost.
- **Number of services** — order multiple identical servers at once (up to 10).

The total cost shown in the summary reflects all selected options.

To proceed, click **Pay** to go directly to payment, or click **To cart** to save the order to the cart for later.

## Server renewal

After the initial paid period ends, the server switches to daily billing automatically. The account balance is charged at a proportional rate — for a 30-day month, 1/30 of the plan cost is charged each day.

## Activation timeline

A server can remain in «Activation in progress» status for 15 minutes to 3 hours, after which it transitions to «Active» status and is ready to use.

A server saved to the cart using **To cart** appears in «Ordered» status until payment is completed. To activate it, verify the account balance and add funds if needed — the server then transitions to «Activation in progress» status.

If the server hasn't transitioned to «Active» status within 3 hours, contact support via chat or ticket from the Control Panel.

Plan changes for virtual servers are handled by the Gcore support team. To switch to a different plan, submit a support ticket — support will apply the change and notify when it is done.

The new plan must match the same data center as the current server, and its vCPU, RAM, disk space, bandwidth, and traffic specifications must be equal to or higher than the current plan.

## Request an upgrade

<Steps>
  <Step title="Find the server ID and current plan">
    Navigate to **Products/Services** → **Virtual private servers** in the left sidebar.

    Note the **Id** and **Tariff plan** columns for the server to upgrade.

    <Frame>![Virtual private servers list showing server ID, tariff plan, data center, expiration date, and service status columns](/images/docs/hosting/virtual-servers/upgrade-your-virtual-server/upgrade-your-virtual-server-image4.png)</Frame>
  </Step>
  <Step title="Select a target plan">
    Navigate to **Products/Services** → **Virtual server (VDS)** to browse the plan catalog.

    Use the **Data center** filter to show only plans available in the same location as the current server. Note the exact plan name from the card header — this is the name to include in the support ticket.

    <Frame>![Virtual server plan catalog filtered by data center, showing plan names with vCPU, RAM, disk, bandwidth, and price](/images/docs/hosting/virtual-servers/upgrade-your-virtual-server/upgrade-your-virtual-server-image5.png)</Frame>
  </Step>
  <Step title="Open Support tickets and click Add">
    Navigate to **Support** → **Support tickets** in the left sidebar, then click **Add**.
  </Step>
  <Step title="Fill in the ticket form">
    Complete the form:

    - **Subject** — describe the request — `Upgrade server 3253148 to KVM-SSD-2-SY4`
    - **Service** — select the server from the dropdown list
    - **Priority** — select the urgency level
    - **Message** — add any additional details or preferred upgrade time

    <Frame>![Support ticket form with Subject, Service dropdown, Priority radio buttons, and message area](/images/docs/hosting/virtual-servers/upgrade-your-virtual-server/upgrade-your-virtual-server-image6.png)</Frame>
  </Step>
  <Step title="Submit the ticket">
    Click **Submit**. The support team will confirm the request and schedule the upgrade.
  </Step>
</Steps>

## After the upgrade

- **Server data and IP address are preserved.** No data is deleted and the server IP does not change.
- **Downtime is 30–60 minutes.** During this window the server is unavailable while settings are applied and it reboots.
- **Upgrades run on weekdays.** Tickets can be submitted any day; the time can be agreed in the ticket chat.
- **The plan change is permanent.** The previous plan cannot be restored.
- **New disk space is not mounted automatically.** If the new plan has more disk space, expand the logical partitions inside the OS or create a new partition manually.
- **Pricing adjusts from the upgrade date.** If the server's minimum order period (shown in the **Expiration date** column) has not expired, the prepaid balance is recalculated proportionally. A server ordered at 20 EUR/month with 60 EUR paid for three months, then upgraded to 40 EUR/month immediately, leaves a balance that covers only one and a half months at the new rate.

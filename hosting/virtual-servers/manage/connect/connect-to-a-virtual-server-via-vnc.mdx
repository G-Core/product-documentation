---

VNC (Virtual Network Computing) enables remote desktop control by transmitting display output, mouse, and keyboard input between client and server using the Remote Frame Buffer protocol. Access the VNC console directly from the [hosting panel](https://hosting.gcore.com/billmgr) to connect to a virtual server.

<Steps>
  <Step title="Find the credentials">
    Note the **User** and **Password** for the server. These are included in the activation email and are also available in the hosting panel — click **Instructions** in the toolbar to view them.

    <Frame>![Virtual server activation page showing User and Password fields](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image1.png)</Frame>
  </Step>
  <Step title="Open VMmanager">
    In the hosting panel, select **Products/Services**, click **Virtual private servers**, and click **To panel**.

    <Frame>![Virtual private servers list with the To panel button in the toolbar](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image3.png)</Frame>
  </Step>
  <Step title="Open the VNC console">
    In VMmanager, click **VNC** in the toolbar.

    <Frame>![VMmanager toolbar showing the VNC button](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image4.png)</Frame>
  </Step>
  <Step title="Log in to the server">
    In the VNC console, enter the **User** and **Password** from the activation email or the **Instructions** page.

    <Frame>![VNC console showing Ubuntu login prompt](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image5.png)</Frame>
  </Step>
</Steps>

The virtual server desktop is now accessible from the VNC console.

The VNC console in VMmanager 6 provides browser-based access to a virtual server without installing a separate client.

## Connection setup

<Steps>
  <Step title="Retrieve connection credentials">
    Log in to the [Hosting&nbsp;portal](https://hosting.gcore.com). In the left sidebar, select **Products/Services** → **Virtual private servers**. Check the box next to the server, then click **Instructions** in the toolbar.

    <Frame>![Virtual private servers list with a server selected and the Instructions button in the toolbar](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image1.png)</Frame>

    A page opens listing the server **IP address**, **Username**, and **Password**. The same credentials are in the activation email sent after provisioning.
  </Step>
  <Step title="Open VMmanager">
    In the **Virtual private servers** list, check the box next to the server, then click **To panel** in the toolbar.

    <Frame>![Virtual private servers list with a server selected and the To panel button in the toolbar](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image3.png)</Frame>

    VMmanager 6 opens on the server's information page.
  </Step>
  <Step title="Start the VNC session">
    Click **VNC** in the top right corner, or open **Menu** and select **VNC**.

    <Frame>![VMmanager server page with the Menu dropdown open and VNC option visible](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image4.png)</Frame>

    The VNC console opens in the browser.
  </Step>
  <Step title="Authenticate">
    At the login prompt, enter the **Username** and **Password** from the Instructions page or the activation email.

    <Frame>![VNC console showing Ubuntu login prompt with root user authenticated](/images/docs/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc/connect-to-a-virtual-server-via-vnc-image5.png)</Frame>
  </Step>
</Steps>

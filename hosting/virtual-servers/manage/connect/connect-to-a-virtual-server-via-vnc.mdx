title: Connect to a virtual server via VNC in VMmanager 6
sidebarTitle: VNC
ai-navigation: Connect to a virtual server via VNC console in VMmanager 6 using the Remote Frame Buffer protocol and root credentials from the Instructions tab.
---

VNC (Virtual Network Computing) lets you view and control a remote computer's desktop. It works by transmitting graphics, mouse, and keyboard input between client and server applications using the Remote Frame Buffer protocol. You can use the VNC console to connect to your virtual server directly from the control panel.

1. After activating your virtual server, you'll receive an email with connection instructions and login credentials. This information is also available in the control panel on the **Instructions** tab, where you'll find your **User** and **Password**.

<Frame>![Instructions tab in the control panel](/images/docs/hosting/virtual-servers/manage/connect/connect-to-your-virtual-server-via-vnc-in-vmmanager-6/1-intsructions-in-the-menu.png)</Frame>

<Frame>![Instructions in the activation email](/images/docs/hosting/virtual-servers/manage/connect/connect-to-your-virtual-server-via-vnc-in-vmmanager-6/2-instructions-in-the-email.png)</Frame>

2. Open the control panel. Select **Products/Services**, click **Virtual private servers**, and then click **To Panel**.

<Frame>![To Panel button in the control panel](/images/docs/hosting/virtual-servers/manage/connect/connect-to-your-virtual-server-via-vnc-in-vmmanager-6/4-to-panel-new-interface.png)</Frame>

3. In the VMmanager panel, open the three-dot menu and click **VNC**.

<Frame>![VNC option in VMmanager 6](/images/docs/hosting/virtual-servers/manage/connect/connect-to-your-virtual-server-via-vnc-in-vmmanager-6/5-vnc-in-vmmanager-6.png)</Frame>

The VNC server management console will open. Enter the root login and password from your activation email or the **Instructions** tab.

<Frame>![VNC server management console](/images/docs/hosting/virtual-servers/manage/connect/connect-to-your-virtual-server-via-vnc-in-vmmanager-6/6-vnc.png)</Frame>

You've successfully connected to your server using VNC.

SSH keys enable password-free authentication to virtual servers. Manage them from the **SSH-keys** section in VMmanager 6 account settings.

To open the SSH-keys section:

1. Log in to [BillMgr](https://hosting.gcore.com/billmgr) and navigate to **Products/Services** → **Virtual private servers**.

2. Select the server row and click **To panel** — VMmanager 6 opens in a new tab.

3. Click the user avatar in the top-right corner and select **SSH-keys**.

<Frame>![SSH-keys section in VMmanager 6 showing the empty state with the Add an SSH-key button](/images/docs/hosting/virtual-servers/manage/connect/manage-ssh-keys/manage-ssh-keys-image1.png)</Frame>

**To add a key:**

1. Click **Add** or **Add an SSH-key**.

2. In the **Name** field, enter a name for the key.

3. In the **SSH-key content** field, paste the public key.

4. Click **Add**.

<Frame>![Add an SSH-key dialog in VMmanager 6 with Name and SSH-key content fields](/images/docs/hosting/virtual-servers/manage/connect/manage-ssh-keys/manage-ssh-keys-image2.png)</Frame>

**To rename a key:**

1. Click the edit icon next to the key.

2. In the **Name** field, enter the new name and click **Save**.

The key content cannot be changed after a key is added. To use a different public key, delete the existing entry and add a new one.

**To delete a key:**

1. Click the delete icon next to the key.

2. In the confirmation dialog, click **Delete the SSH-key**.

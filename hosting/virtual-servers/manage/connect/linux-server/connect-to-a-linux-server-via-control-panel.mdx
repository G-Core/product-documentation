---
title: Connect to a Linux server via control panel
ai-navigation: Access a Linux virtual server terminal console through the VMmanager control panel browser-based VNC session.
---

VMmanager provides browser-based console access to Linux servers without requiring an SSH client or a separate VNC application. Use it whenever direct SSH access is unavailable or when troubleshooting networking and boot issues.

<Steps>
  <Step title="Open the server in VMmanager">
    Log in to the [Hosting&nbsp;portal](https://hosting.gcore.com) and navigate to **Products/Services** > **Virtual private servers**.

    Select the server row by checking its checkbox, then click **To panel** in the toolbar. VMmanager opens in a new tab.

    <Frame>
      ![Virtual private servers list with a server selected and the To panel button highlighted in the toolbar](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/connect-to-a-linux-server-via-control-panel-image1.png)
    </Frame>
  </Step>
  <Step title="Open the VNC console">
    On the VM page, click **VNC** in the top right corner of the VMmanager header.

    <Frame>
      ![VMmanager VM detail page with VNC button in the top right header](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/connect-to-a-linux-server-via-control-panel-image2.png)
    </Frame>

    A browser-based console session opens in a new window.
  </Step>
  <Step title="Log in to the server">
    At the login prompt, enter the **Username** and **Password** for the server. The credentials appear in the activation email sent after provisioning and in the **Instructions** tab in the Hosting portal toolbar.

    <Frame>
      ![VMmanager VNC console showing a Linux login prompt with username and password fields](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/connect-to-a-linux-server-via-control-panel-image3.png)
    </Frame>
  </Step>
</Steps>

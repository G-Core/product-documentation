---
title: Connect to a Linux server via Control Panel
sidebarTitle: Connect to a Linux server via Control panel
---

  1. Open the list of servers, click on the server, and press "To panel" (you will open VMmanager).


<Frame>![list of servers](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/mceclip2.png)</Frame>



2\. Click the monitor icon.


<Frame>![monitor icon](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/mceclip3.png)</Frame>



The panel is open! You can control the server remotely using it.

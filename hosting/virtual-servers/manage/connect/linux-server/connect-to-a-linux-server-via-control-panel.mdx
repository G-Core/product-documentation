---
title: Connect to a Linux server via control panel
ai-navigation: Connect to a Linux server console remotely using the VMmanager VNC console, accessed from the BillMgr hosting control panel.
---

The VMmanager VNC console connects directly to a virtual server's display, allowing remote access even when SSH is unavailable.

<Steps>
  <Step title="Open the server list">
    Navigate to **Products/Services** → **Virtual private servers**. Select the server and click **To panel** in the toolbar.

    <Frame>
      ![Virtual private servers list in BillMgr with a server selected and the To panel button highlighted in the toolbar](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/connect-to-a-linux-server-via-control-panel-image1.png)
    </Frame>
  </Step>
  <Step title="Click the server name in VMmanager">
    A new tab opens. Click the server name to expand its details.

    <Frame>
      ![VMmanager VM list with an expanded server row showing the VNC button](/images/docs/hosting/virtual-servers/manage/connect/linux-server/connect-to-a-linux-server-via-control-panel/connect-to-a-linux-server-via-control-panel-image2.png)
    </Frame>
  </Step>
  <Step title="Open the VNC console">
    Click **VNC**. The console opens in a new tab for remote server access.
  </Step>
</Steps>

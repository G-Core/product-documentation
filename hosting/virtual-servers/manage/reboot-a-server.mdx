You can reboot your virtual private server from two interfaces: BILLmanager and VMmanager.

## Via BILLmanager

1. In BILLmanager, go to **Virtual private servers**.
2. Find the server you want to reboot, click **...** at the end of its row, and select **Reboot**.

<Frame>
  ![The Virtual private servers list in BILLmanager with the context menu open and the Reboot option highlighted](/images/docs/hosting/virtual-servers/manage/reboot-a-server/reboot-a-server-image1.png)
</Frame>

## Via VMmanager

1. In VMmanager, go to **Virtual machines**.
2. Find the server you want to reboot, click **...** at the end of its row, and select **Restart**.

<Frame>
  ![The Virtual machines list in VMmanager with the context menu open and the Restart option highlighted](/images/docs/hosting/virtual-servers/manage/reboot-a-server/reboot-a-server-image2.png)
</Frame>

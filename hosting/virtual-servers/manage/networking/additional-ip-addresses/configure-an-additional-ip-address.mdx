An additional IP address purchased in the control panel becomes active only after it is configured in the operating system.

To find the netmask for the purchased address, navigate to the **Virtual private servers** or **Dedicated servers** tab in the Gcore Hosting [control&nbsp;panel](https://hosting.gcore.com/billmgr). Select the server and click **IP addresses**.

## Debian and Ubuntu

Edit the network interfaces file to add the IP address.

1. Open `/etc/network/interfaces`.
2. Add the following configuration. Replace `N` with the sequential index of the additional IP address (starting at `0` for the first), `X.X.X.X` with the IP address, and `255.255.255.255` with the actual netmask:

```sh
auto eth0:N
iface eth0:N inet static
    address X.X.X.X
    netmask 255.255.255.255
```

(Optional) To add an IPv6 address, also append:

```sh
iface eth0:N inet6 static
    address 2001:X:X:X::X
```

3. Save the file.
4. Restart networking: run `systemctl restart networking`.

## CentOS

Create a per-interface configuration file for each additional IP address.

1. Create the file `/etc/sysconfig/network-scripts/ifcfg-eth0:N`.
2. Add the following configuration. Replace `N` with the sequential index of the additional IP address (starting at `0` for the first), `X.X.X.X` with the IP address, and `255.255.255.255` with the actual netmask:

```sh
DEVICE=eth0:N
ONBOOT=yes
BOOTPROTO=static
IPADDR=X.X.X.X
NETMASK=255.255.255.255
```

(Optional) To add an IPv6 address, also set:

```sh
IPV6ADDR_SECONDARIES="2001:X:X:X::X/64"
```

3. Save the file.
4. Restart networking: run `systemctl restart network`.

## Windows Server

Configure additional IP addresses through the network adapter properties.

1. Connect to the server via [RDP](/hosting/dedicated-servers/manage/connect/connect-to-a-windows-server).
2. Click **Start** and open **Server Manager**.
3. In the Server Manager window, click **Local Server**.
4. Right-click the network connection and select **Properties**.
5. Select the IP version: **Internet Protocol Version 4 (TCP/IPv4)** for IPv4 or **Internet Protocol Version 6 (TCP/IPv6)** for IPv6.
6. Click **Advanced**, then **Add** to enter the IP address and subnet mask.
7. Click **OK**, then **Close**.

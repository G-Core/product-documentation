Windows Server is available for KVM-SSD-4 and KVM-HIGHCPU-SSD-4 plans and higher. The license adds 25.00 EUR to the monthly cost.

## Order a Windows Server VPS

<Steps>
<Step title="Open the plan list">

In the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), navigate to **Virtual private servers** and click **Order**.

</Step>
<Step title="Find a plan that supports Windows Server">

Use the **Data center** filter to select a region. KVM-SSD-4 and KVM-HIGHCPU-SSD-4 plans and higher display the **Operating systems: Free OS** tag — these support Windows Server.

<Frame>![VPS plan list filtered by Mexico, Santiago de Querétaro, showing KVM-HIGHCPU-SSD-4-SQR and KVM-HIGHCPU-SSD-5-SQR plans with the Operating systems Free OS tag](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image1.png)</Frame>

</Step>
<Step title="Open the order form">

On a KVM-SSD-4 or KVM-HIGHCPU-SSD-4 plan card (or higher), click **Order**.

</Step>
<Step title="Select Windows Server">

In the **Operating system** field, select **Windows**, then select a version.

<Frame>![KVM-HIGHCPU-SSD-4-SQR order form with Windows selected, showing Windows Server 2019 and Windows Server 2022 options in the dropdown](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image2.png)</Frame>

</Step>
<Step title="Complete the order">

Configure the remaining options — payment period, number of public IPv4 addresses, and IPv6 network — then click **Registration and payment**.

</Step>
</Steps>

## Windows Server notes

Keep these constraints in mind before ordering.

- Windows Server can only be selected at order time. It cannot be added to a server that was ordered with a Linux OS.
- Switching from Windows to a Linux OS is possible via an [OS&nbsp;template](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template), but the Windows license fee continues to be charged after reinstalling. Switching back to Windows after that is not possible without contacting support.
- Ordering a new server with the required OS avoids both limitations.

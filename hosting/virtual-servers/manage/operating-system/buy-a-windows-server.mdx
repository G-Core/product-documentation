Gcore hosting supports Windows Server 2019, 2022, and 2025 as an operating system option on Virtual Server (VDS) plans.

<Note>Windows Server is available on KVM-SSD-4 plans and higher.</Note>

## Buy a Windows server

Ordering a Windows Server VDS involves selecting a compatible plan from the catalog and selecting Windows on the configuration form.

1. [Log in](https://hosting.gcore.com/billmgr) to the Gcore hosting control panel. If no account exists yet, [sign up](https://hosting.gcore.com/billmgr?func=register&lang=en) first.

2. On the **Home** page, click **Order a service**.

<Frame>![Home page with Order a service button](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image1.png)</Frame>

3. Click **Virtual server (VDS)**.

<Frame>![Order service catalog showing Virtual server (VDS) section](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image2.png)</Frame>

4. Use the **Data center** filter to select a location.

<Frame>![Data center filter dropdown with all available locations](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image3.png)</Frame>

5. Use the **Payment period** filter to select a billing cycle: **Month**, **Three months**, **Half a year**, or **Year**.

<Frame>![Payment period dropdown showing monthly, quarterly, semi-annual, and annual options](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image4.png)</Frame>

6. Find a plan with KVM-SSD-4 or higher and click **Order**.

7. On the configuration form, under **Operating system**, select **Windows**, then select a version from the dropdown.

<Frame>![Windows Server selected in the Operating system field on the server configuration form](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image5.png)</Frame>

8. Review the order summary and click **Pay** or **To cart**.

## Features of a Windows server

Two constraints apply to Windows Server VDS instances.

- Windows can be installed only when ordering a server — it cannot be added after the server is provisioned.

- Windows can be replaced with another OS [using templates](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template).

Windows Server is available on KVM-SSD-4 and KVM-HIGHCPU-SSD-4 plans and higher. The license adds EUR 25.00 to the monthly cost.
<Steps>
<Step title="Open the plan list">

In the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), navigate to **Virtual private servers** and click **Order**.

</Step>
<Step title="Find a plan that supports Windows Server">

Use the **Data center** filter to select a region. KVM-SSD-4 and KVM-HIGHCPU-SSD-4 plans and higher display the **Operating systems: Free OS** tag — these support Windows Server. Click **Order** on the plan.

<Frame>![VPS plan list in Mexico filtered by data center, showing KVM-HIGHCPU-SSD-4-SQR with the Operating systems Free OS tag](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image1.png)</Frame>

</Step>
<Step title="Select Windows Server">

In the **Operating system** field, select **Windows**, then select a version.

<Frame>![KVM-HIGHCPU-SSD-4-SQR order form with Windows selected and version dropdown expanded](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image2.png)</Frame>

</Step>
<Step title="Complete the order">

Configure the remaining options, then click **Registration and payment**.

</Step>
</Steps>

## Windows Server notes

Keep these constraints in mind before ordering.

- Windows Server can only be selected at order time. It cannot be added to a server that was ordered with a Linux OS.
- Reinstalling with Linux via an [OS&nbsp;template](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template) is possible, but the Windows license fee continues to be charged. Reinstalling Windows afterward requires contacting support.

<Note>To avoid reinstallation constraints, create separate servers for Windows and Linux rather than reinstalling the same server.</Note>

Windows Server is available as an operating system for Virtual Private Servers in the KVM-SSD-4 plan and higher. The license adds 25.00 EUR to the monthly cost. Only one data center currently offers Windows Server for VPS: **Mexico, Santiago de Querétaro**.

## Order a Windows Server VPS

<Steps>
<Step title="Open the plan list">

In the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), navigate to **Virtual private servers** and click **Order**.

</Step>
<Step title="Filter by data center">

Click **Data center**, select **Mexico, Santiago de Querétaro**, then click **Select**.

Plans that support Windows Server display the **Operating systems: Free OS** tag — these are KVM-SSD-4 and higher.

<Frame>![VPS plan list filtered by Mexico, Santiago de Querétaro, showing kvm-ssd-4 and higher plans with the Operating systems Free OS tag](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image1.png)</Frame>

</Step>
<Step title="Open the order form">

On a KVM-SSD-4 or higher plan card, click **Order**.

</Step>
<Step title="Select Windows Server">

In the **Operating system** field, click **Windows**, then select a version from the dropdown: **Windows Server 2019** or **Windows Server 2022**.

The price summary on the right updates to show the Windows Server license fee under **For the configuration**.

<Frame>![Order form for KVM-SSD-4-SQR with Windows Server 2019 selected, showing 25.00 EUR license fee and 44.99 EUR total](/images/docs/hosting/virtual-servers/manage/operating-system/buy-a-windows-server/buy-a-windows-server-image2.png)</Frame>

</Step>
<Step title="Complete the order">

Configure the remaining options — payment period, number of public IPv4 addresses, and IPv6 network — then click **Registration and payment**.

</Step>
</Steps>

## Windows Server notes

Keep these constraints in mind before ordering.

- Windows Server can only be installed at order time. It cannot be added to a server that was ordered with a Linux OS.
- Switching from Windows to a Linux OS is possible via an [OS&nbsp;template](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template), but the Windows license fee continues to be charged after reinstalling. Ordering a new server with the required OS is the recommended approach instead.

VMmanager 6 supports installing an operating system from a custom ISO image by uploading the file and connecting it to a virtual server as a bootable disk.

<Warning>
When **Start VM from ISO image** is enabled, VMmanager restarts the VM automatically after the ISO is connected. Back up any data on the server before proceeding.
</Warning>

To install a Linux OS from an ISO image:

1. Log in to [BillMgr](https://hosting.gcore.com/billmgr) and navigate to **Products/Services** > **Virtual private servers**.

2. Select the server row and click **To panel** — VMmanager opens in a new tab.

<Frame>![Virtual machines list in VMmanager 6 showing one active VM](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image1.png)</Frame>

3. Click the VM name to open its detail page, then click the **Virtual disk** tab.

<Frame>![Virtual disk tab in VMmanager 6 showing VM virtual disks section with Connect ISO images link](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image2.png)</Frame>

4. Click **Connect ISO images** — a side panel opens.

<Frame>![Connect ISO images side panel showing Upload method, file upload area, Start VM from ISO image checkbox, and Connect button](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image3.png)</Frame>

5. In the **Upload method** field, select **Local file**, then drag and drop or select the ISO file. The maximum file size is 8.6 GB.

6. Keep **Start VM from ISO image** checked — VMmanager sets the boot priority to the ISO and restarts the VM automatically after the upload.

7. Click **Connect**.

8. When the VM restarts, click **VNC** to open the console and complete the OS installation from the boot menu.

Gcore also supports [template installation](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template) for servers that need a pre-configured operating system.

VMmanager 6 supports installing an operating system from a custom ISO image. The ISO file is downloaded from a public URL and mounted to the server as a bootable disk.

## ISO requirements

The ISO image must meet these requirements:

- The file is accessible via a public HTTP or HTTPS URL
- Maximum file size: 8 GB (8,192 MB)
- Only one ISO image can be connected to a server at a time

<Warning>
Installing from ISO erases all data on the server disk. Back up any important files before proceeding.
</Warning>

## Mount the ISO image

<Steps>
  <Step title="Open the Hosting portal">
    Log in to the [Hosting portal](https://hosting.gcore.com/billmgr) and navigate to **Products/Services** → **Virtual private servers**.
  </Step>
  <Step title="Open VMmanager 6">
    Select the checkbox next to the server, then click **To panel** in the toolbar.

    <Frame>![Virtual private servers list with a server row selected and the To panel button highlighted in the toolbar](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image1.png)</Frame>
  </Step>
  <Step title="Open the Mount ISO-image menu">
    In VMmanager 6, go to the **Virtual machines** list. Open the three-dot menu on the right side of the server row and select **Mount ISO-image**.

    <Frame>![VMmanager 6 virtual machines list with the three-dot menu open and Mount ISO-image option highlighted](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image2.png)</Frame>
  </Step>
  <Step title="Enter the ISO image URL">
    In the **URL** field, enter a direct link to the ISO file. Optionally, add **Tags** — tag values are matched against VMmanager script tags to trigger automated setup scripts after installation.

    <Frame>![Upload ISO-image page with URL field and Tags field](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image3.png)</Frame>
  </Step>
  <Step title="Download and connect">
    Click **Download and connect**. VMmanager 6 downloads the ISO image, mounts it as a bootable disk, and reboots the server from it automatically.
  </Step>
  <Step title="Complete the installation via VNC">
    After the server reboots, connect via [VNC](/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc) to reach the installer. The ISO boot menu appears in the console — follow the installer steps to complete the OS installation.
  </Step>
</Steps>

<Note>
After the installation is complete, disconnect the ISO image to restore normal boot: open **Virtual disk** in the left sidebar, remove the ISO entry, and restart the server.
</Note>

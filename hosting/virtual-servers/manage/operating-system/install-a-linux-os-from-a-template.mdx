---
title: Install a Linux OS from a template
sidebarTitle: Install Linux from a template
ai-navigation: Reinstall Linux operating systems from templates including CentOS 7, Debian 8/10, FreeBSD 10/11/12, and Ubuntu 14.04/16.04/18.04/20.04 via VM Manager Control panel Reinstall button.
---

Go to the VM Manager Control panel. Or, in the main control panel, under "Virtual servers", select your server and click "To panel".


<Frame>![VM Manager Control panel](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template/install-a-linux-os-from-a-template-image1.png)</Frame>



Then go to Virtual Machines and choose a server and press the Reinstall button.


<Frame>![Virtual Machines tab](/images/docs/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-a-template/install-a-linux-os-from-a-template-image2.png)</Frame>



You can choose one of the following operating systems from the list of our templates:

  * CentOS 7-amd64
  * Debian 8-x86_64
  * Debian 10-x86_64
  * FreeBSD 10-amd64
  * FreeBSD 11-amd64
  * FreeBSD 12-amd64 
  * Ubuntu 14.04-amd64
  * Ubuntu 16.04-amd64
  * Ubuntu 18.04-amd64
  * Ubuntu 20.04-amd64



If you purchase Windows 2012, you can reinstall it from the VM Manager control panel too.

You can install any other operating system except Windows (Windows could be purchased only at the time of ordering server and cannot be installed later) on your machine for free. To do this, you only need an operating system image in the [.iso format](/hosting/virtual-servers/manage/operating-system/install-a-linux-os-from-your-iso-image).

---

[VMmanager&nbsp;6](/hosting/virtual-servers/manage/main-features-and-functions-of-vmmanager-6) supports installing an operating system from a custom ISO image. Provide a public HTTP, HTTPS, or FTP URL to an ISO file (up to 8 GB). Only one ISO image can be mounted to a server at a time.

<Note>
During ISO installation, the following operations are unavailable: root password editing, creating ISO images, and running automation scripts while the ISO is loading.
</Note>

<Steps>
  <Step title="Open VMmanager 6">
    Log in to the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), navigate to **Products/Services** → **Virtual private servers**, select the checkbox next to the server, and click **To panel**.

    <Frame>![BILLmanager Virtual private servers list with a server row selected and To panel button highlighted in the toolbar](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-image/install-an-os-from-iso-in-vmmanager-6-image1.png)</Frame>
  </Step>
  <Step title="Open the ISO mount menu">
    In VMmanager, open the three-dot menu next to the server and select **Mount ISO-image**.

    <Frame>![VMmanager Virtual machines list with the three-dot menu open and Mount ISO-image highlighted](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-image/install-an-os-from-iso-in-vmmanager-6-image2.png)</Frame>
  </Step>
  <Step title="Enter the ISO URL">
    Enter the URL where the ISO image is hosted. Optionally, add **Tags** to trigger VMmanager automation scripts that have matching tag values.

    <Frame>![Mount ISO-image dialog with URL field and Tags field](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-image/install-an-os-from-iso-in-vmmanager-6-image3.png)</Frame>
  </Step>
  <Step title="Mount and install">
    Click **Download and connect**. VMmanager downloads the ISO and restarts the server from it. Connect via [VNC](/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc) to complete the OS installation.
  </Step>
</Steps>

An ISO image is a disk file image containing an exact copy of the data and file structure from an optical disc such as a CD, DVD, or Blu-ray. ISO images can include operating systems, programs, or any other data that can be written to an optical disc.

## ISO requirements

The ISO file must meet the following requirements:

- Maximum image size: ~8 GB (8,192 MB)
- Only one ISO image can be uploaded at a time

## ISO restrictions

While installing an OS from an ISO image, the following actions are unavailable:

- Root password editing
- Creating new ISO images
- Running scripts on the VM while the ISO image is loading

## Install an OS from an ISO image

Connect an ISO image to a virtual server by providing a URL and configuring the VM's boot settings in VMmanager.

1. Open the control panel. Select **Products/Services**, click **Virtual private servers**, and then click **To Panel**.

<Frame>
  ![Virtual private servers list in BillMgr with the To panel button highlighted](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-in-vmmanager-6/install-an-os-from-iso-in-vmmanager-6-image1.png)
</Frame>

2. In the VMmanager panel, click the three-dot menu on the VM row and select **ISO images**.

<Frame>
  ![ISO images option in the VMmanager three-dot menu](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-in-vmmanager-6/install-an-os-from-iso-in-vmmanager-6-image2.png)
</Frame>

3. In the **Upload method** dropdown, select **Image URL**.

4. Fill in the form:

   - **Image URL** (required): Enter the URL of the ISO file.
   - **OS type**: Select the operating system family — **Linux** (default), **Windows**, **FreeBSD**, or **Other OS**. The image is tagged with the selected type, which determines which scripts can run on the VM.
   - **Image tags** (optional): Enter additional comma-separated tags.
   - **Start VM from ISO image**: Keep this checkbox selected (default) to boot the VM from the ISO image immediately. The VM's boot priority is set to ISO and the VM restarts automatically.

<Frame>
  ![Connect ISO images panel with Image URL, OS type, and Image tags fields](/images/docs/hosting/virtual-servers/manage/operating-system/install-an-os-from-iso-in-vmmanager-6/install-an-os-from-iso-in-vmmanager-6-image3.png)
</Frame>

5. Click **Connect**.

After the VM restarts, access it via [VNC](/hosting/virtual-servers/manage/connect/connect-to-a-virtual-server-via-vnc).

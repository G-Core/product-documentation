Order a dedicated server by selecting a data center and server configuration, configuring the order parameters, and completing payment. The server is typically ready within 15 minutes to 3 hours after payment.

## Tariff selection

Tariffs are available in two places: the [Control&nbsp;Panel](https://hosting.gcore.com/billmgr) and the [Gcore&nbsp;website](https://gcore.com/hosting). Both provide the same catalog of dedicated servers.

In the Control Panel, use the **Data center** filter to select a location. On the website, use the configurator to filter servers by technical characteristics.

  ![Control Panel catalog showing available dedicated servers filtered by data center](/images/docs/hosting/dedicated-servers/order-a-dedicated-server/order-a-dedicated-server-image1.png)

  ![Gcore website server configurator with tabs and filter panel](/images/docs/hosting/dedicated-servers/order-a-dedicated-server/order-a-dedicated-server-image2.png)

Select the preferred data center and tariff, then click **Order** (Control Panel) or **Buy** (website).

If none of the available tariffs fit the required configuration, [contact&nbsp;support](/hosting/contact-our-technical-support) with the number of servers needed and the desired configuration.

To order servers in multiple data centers, create a separate order for each data center.

## Order parameters

The service order page provides the following configuration options:

<Frame>
  ![Service order page with domain, OS, payment period, and IP address fields](/images/docs/hosting/dedicated-servers/order-a-dedicated-server/order-a-dedicated-server-image4.png)
</Frame>

| Parameter | Description |
|---|---|
| **Domain name** | Enter a domain name for the server |
| **Operating system** | Select from the available list. It can be reinstalled later from a template or a custom ISO image |
| **Traffic** | Select the included traffic package (Dedicated Servers only) |
| **Additional IP addresses** | Order up to 15 IPv4/IPv6 addresses for a Dedicated Server, or up to 10 for a virtual one |

<Note>KVM-SSD-1 servers support only three additional IPs of each type.</Note>

## Payment

Click **Add to cart** to proceed to the cart, then click **Pay**. Select a payment method from the available options and complete the payment. The purchased server will appear in the **Products/Services** section.

## Activation

After payment, the server enters activation mode and becomes available within 15 minutes to 3 hours.

<Warning>If the server is not activated after 3 hours, [contact&nbsp;support](/hosting/contact-our-technical-support).</Warning>

## Custom server configurations

If none of the available configurations meet the requirements, [contact&nbsp;support](/hosting/contact-our-technical-support) to discuss a custom server configuration.

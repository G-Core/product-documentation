Deleting a dedicated server removes it from the server list, erases all stored data, and stops billing charges.

<Warning>
Deleted servers cannot be restored.
</Warning>

<Steps>
  <Step title="Open the dedicated servers list">
    Log in to the [Gcore&nbsp;Hosting&nbsp;Portal](https://hosting.gcore.com/billmgr) and navigate to **Products/Services** > **Dedicated servers**.

    <Frame>
      ![Dedicated servers list with a server row selected and the Delete button visible](/images/docs/hosting/dedicated-servers/delete-a-dedicated-server/delete-a-dedicated-server-image1.png)
    </Frame>
  </Step>
  <Step title="Confirm the deletion">
    Select the server to delete and click **Delete**. In the confirmation dialog, click **OK**.

    <Frame>
      ![Confirmation dialog for deleting a dedicated server with OK and Cancel buttons](/images/docs/hosting/dedicated-servers/delete-a-dedicated-server/delete-a-dedicated-server-image2.png)
    </Frame>
  </Step>
</Steps>

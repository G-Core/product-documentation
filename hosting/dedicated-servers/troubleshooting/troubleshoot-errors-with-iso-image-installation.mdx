## ISO upload fails in DCI manager

To upload an ISO image, navigate to **Products/Services** > **Dedicated servers** in the [Hosting&nbsp;portal](https://hosting.gcore.com/billmgr), select a server, and click **To Panel** to open DCI manager.

If the upload fails, verify that the ISO file meets all of the following requirements:

- **File extension:** must be `.iso` (lowercase only — `.ISO` is not accepted)
- **File name:** letters and digits only; no spaces, hyphens, or special characters
- **File size:** must not exceed 9 GB
- **Account limit:** a maximum of four ISO images are allowed per account

## ISO mounting error in DCI manager

If DCI manager returns the following error when mounting an ISO image:

```
Error occurred when mounting ISO-image on the IPMI proxy server. Script output: ''
```

Delete the ISO image from DCI manager and upload it again. If the error persists, contact [support](https://hosting.gcore.com/billmgr?startform=clientticket.edit) via a support ticket.

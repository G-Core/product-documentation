All IP addresses assigned to servers are geolocated to the same city as the data center. To verify an IP address location, check it against these two authoritative databases:

- [MaxMind&nbsp;GeoIP](https://www.maxmind.com/en/geoip-demo)
- [RIPEstat](https://stat.ripe.net)

Other online services may display an outdated location because they update geolocation data with a delay.

If the location shown by MaxMind or RIPEstat is incorrect, contact [technical support](/hosting/contact-our-technical-support) — Gcore will submit a location correction request to the geolocation providers on behalf of the account.

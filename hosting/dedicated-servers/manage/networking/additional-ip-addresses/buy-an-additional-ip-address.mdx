Every dedicated server includes one free IPv4 address and one free IPv6 address. Up to 14 additional IPv4 and 14 additional IPv6 addresses are available for purchase.

<Note>
For KVM-SSD-1 virtual private servers, the limit is 2 additional IPv4 addresses and 2 additional IPv6 addresses.
</Note>

1. Navigate to the **Dedicated Servers** or **Virtual private servers** tab in the Gcore Hosting [control&nbsp;panel](https://hosting.gcore.com/billmgr).
2. Select the server and click **IP addresses**.

<Frame>
  ![Dedicated servers list with the IP addresses button highlighted in the toolbar](/images/docs/hosting/dedicated-servers/manage/networking/additional-ip-addresses/buy-an-additional-ip-address/buy-an-additional-ip-address-image1.png)
</Frame>

3. Click **Order**.

<Frame>
  ![IP addresses management page with the Order button in the toolbar](/images/docs/hosting/dedicated-servers/manage/networking/additional-ip-addresses/buy-an-additional-ip-address/buy-an-additional-ip-address-image2.png)
</Frame>

4. Enter the number of IP addresses and complete the order.

After the order is complete, configure the IP addresses on the server. To configure a large number of addresses automatically, reinstall the operating system.

## Moving IP addresses between servers

Only additional IP addresses can be moved between servers — the primary IP address stays associated with the server. Transfers are a paid service. To request a transfer, contact [support](/hosting/contact-our-technical-support).

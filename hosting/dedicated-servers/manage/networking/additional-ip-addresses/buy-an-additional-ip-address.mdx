Every dedicated server includes one free IPv4 address and one free IPv6 address. Additional IP addresses can be ordered at any time after the server is created. Up to 14 additional IPv4 and 14 additional IPv6 addresses are available for purchase.

<Note>
For KVM-SSD-1 virtual private servers, the limit is 2 additional IPv4 addresses and 2 additional IPv6 addresses.
</Note>

<Steps>
  <Step title="Open the IP addresses list">
    Navigate to the **Dedicated Servers** or **Virtual private servers** tab in the Gcore Hosting [control&nbsp;panel](https://hosting.gcore.com/billmgr). Select the server and click **IP addresses**.

    <Frame>
      ![Dedicated servers list with the IP addresses button highlighted in the toolbar](/images/docs/hosting/dedicated-servers/manage/networking/additional-ip-addresses/buy-an-additional-ip-address/buy-an-additional-ip-address-image1.png)
    </Frame>
  </Step>
  <Step title="Place the order">
    Click **Order**.

    <Frame>
      ![IP addresses management page with the Order button in the toolbar](/images/docs/hosting/dedicated-servers/manage/networking/additional-ip-addresses/buy-an-additional-ip-address/buy-an-additional-ip-address-image2.png)
    </Frame>
  </Step>
  <Step title="Complete the order">
    Enter the number of IP addresses and complete the order.

    After the order is processed, the new IP addresses appear in the IP address list. Configure them in the operating system before using them.

    To configure a large number of addresses automatically, reinstall the operating system — the reinstallation process applies the full network configuration, including all assigned IP addresses.
  </Step>
</Steps>

## Transfer IP addresses

Additional IP addresses can be transferred between servers. Only additional IP addresses can be moved — the primary IP address stays associated with the server. To request a paid transfer, contact [support](/hosting/contact-our-technical-support).

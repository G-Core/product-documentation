A VLAN (virtual local area network) allows servers to communicate over an internal network regardless of their physical locations. To connect dedicated servers in a VLAN, use DCImanager.

## Create a VLAN

Use DCImanager to create a VLAN and assign it a name.

1. In DCImanager, navigate to the **VLAN** section.

2. Click **Add** to create a new VLAN.

3. Enter a name for the VLAN and click **OK**.

## Add VLAN members

A VLAN can only include servers belonging to the same account.

1. Select the VLAN and navigate to **VLAN Members**.

2. Click **Add**.

3. From the dropdown, select the server to add.

4. In the **Port** section, select a network interface for the server and click **OK**.

The VLAN is ready to use. Any number of servers can be added to a VLAN.

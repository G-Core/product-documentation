VLAN - virtual local network that allows working in an internal network regardless of physical server locations.

In order to set up VLAN between your servers, you should use DCI Manager.

## VLAN initiation

1\. Go to DCI manager and choose VLAN section.


<Frame>![VLAN section.](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid0.png)</Frame>



2\. Click **Add** button to create a new VLAN.


<Frame>![new VLAN](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid1.png)</Frame>



3\. Type a name for VLAN. For example, MyVLAN. Click ОК. 


<Frame>![Type a name for VLAN](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid5.png)</Frame>



## Adding new members

You should add members to realize an interaction by virtual local network. VLAN can be set up only for servers that belong to one account. 

1\. Highlight a created VLAN and go to VLAN Members section. 


<Frame>![VLAN Members section.](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid6.png)</Frame>



2\. Click **Add**. 


<Frame>![Add](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid7.png)</Frame>



3\. Choose a server you need to add in a drop-down list. 


<Frame>![Choose a server you need to add in a drop-down list.](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid8.png)</Frame>



4\. In a Port section choose an interface of the server and click **OK**. 


<Frame>![ Port section](/images/docs/hosting/dedicated-servers/manage/networking/connect-dedicated-servers-into-a-vlan/blobid9.png)</Frame>



After that VLAN is ready to use. A number of servers that can be added to VLAN is unlimited.

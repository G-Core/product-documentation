A Dedicated Server can be rebooted from two interfaces: the [Gcore&nbsp;Hosting&nbsp;Portal](https://hosting.gcore.com/billmgr) and DCImanager.

## Hosting Portal

Use the context menu in the server list to trigger a reboot.

1. In the Hosting Portal, navigate to **Dedicated servers**.
2. Locate the server in the list, click **...** at the end of its row, and select **Reboot**.

<Frame>
  ![The Dedicated servers list in the Hosting Portal with the context menu open and the Reboot option highlighted](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image1.png)
</Frame>

## DCImanager

Use the context menu in the server list to trigger a restart.

1. In DCImanager, navigate to **Servers**.
2. Locate the server in the list, click **...** at the end of its row, and select **Restart**.

<Frame>
  ![The Servers list in DCImanager with the context menu open and the Restart option highlighted](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image2.png)

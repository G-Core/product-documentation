You can reboot your dedicated server from two interfaces: the Control Panel and DCImanager.

## Via the Control Panel

1. In the Control Panel, go to **Dedicated servers**.
2. Find the server you want to reboot, click **...** at the end of its row, and select **Reboot**.

<Frame>
  ![The Dedicated servers list in the Control Panel with the context menu open and the Reboot option highlighted](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image1.png)
</Frame>

## Via DCImanager

1. In DCImanager, go to **Servers**.
2. Select the server you want to reboot.
3. Click **Reboot** in the toolbar.

<Frame>
  ![The Servers list in DCImanager with a server selected and the Reboot button visible in the toolbar](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image2.png)
</Frame>

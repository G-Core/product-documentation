You can reboot your server from three interfaces: BILLmanager, VMmanager, and DCImanager.

## Via BILLmanager

The steps are the same for virtual private servers and dedicated servers.

1. In BILLmanager, go to **Virtual private servers** or **Dedicated servers**.
2. Find the server you want to reboot, click **...** at the end of its row, and select **Reboot**.

## Via VMmanager

1. In VMmanager, go to **Virtual machines**.
2. Find the server you want to reboot, click **...** at the end of its row, and select **Restart**.

<Frame>
  ![The Virtual machines list in VMmanager with the context menu open and the Restart option highlighted](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image2.png)
</Frame>

## Via DCImanager

1. In DCImanager, go to **Servers**.
2. Select the server you want to reboot.
3. Click **Reboot** in the toolbar.

  ![The Servers list in DCImanager with a server selected and the Reboot button visible in the toolbar](/images/docs/hosting/dedicated-servers/manage/reboot-a-server/reboot-a-server-image4.png)

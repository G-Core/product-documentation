IPMI provides low-level remote access to a dedicated server. Use it to monitor hardware, control server power, and reinstall the OS. IPMI access is available for Dedicated Servers in all locations.

## Access IPMI

<Steps>
<Step title="Open DCImanager">

Navigate to the **Dedicated servers** section in the [Gcore&nbsp;Hosting&nbsp;Portal](https://hosting.gcore.com/billmgr), select an active server, and click **To panel**.

<Frame>
  ![Dedicated servers list with the To panel button highlighted](/images/docs/hosting/dedicated-servers/manage/log-in-to-ipmi/login-to-ipmi-image1.png)
</Frame>

</Step>
<Step title="Open the IPMI interface">

In DCImanager, select the server from the list. In the detail panel that opens, find **IPMI v2(lanplus)** in the **Connections** section and click **Go to**. The IPMI interface opens in a new browser tab.

<Frame>
  ![DCImanager server detail panel showing IPMI connection with Go to link](/images/docs/hosting/dedicated-servers/manage/log-in-to-ipmi/mceclip0.png)
</Frame>

</Step>
<Step title="Log in">

Enter the **Local user name** and **Password**, then click **Log In**.

<Frame>
  ![IPMI login interface with Local user name and Password fields](/images/docs/hosting/dedicated-servers/manage/log-in-to-ipmi/3.png)
</Frame>

</Step>
</Steps>

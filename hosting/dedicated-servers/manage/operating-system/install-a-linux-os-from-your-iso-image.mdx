---
title: Install a Linux OS from your ISO image
sidebarTitle: Install a Linux OS from your ISO image
---

You can install almost any other OS on your DS for free. An exception is Windows, it could be purchased at the time of ordering server only and can not be installed later.

Click "Dedicated servers" at the main dashboard, choose your server and click "To Panel".

At the DCI manager, find ISO-images tab and click **Upload**.


<Frame>![ISO-images tab](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/2018-04-10_13-28-53.png)</Frame>



Then go to the console in the IPMI and choose Virtual Storage tab.


<Frame>![Virtual Storage tab](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/2018-04-10_13-36-21.png)</Frame>



Choose ISO file in the Logical Drive type, click Open to go to the ISO directory. You'll find there the uploaded ISO.


<Frame>![ISO ](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/2018-04-10_13-37-04.png)</Frame>



Choose the ISO you need and then click **Open**.


<Frame>![Choose the ISO](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/2018-04-10_13-37-23.png)</Frame>



Then click **Plug in** and **OK** to finish the procedure.


<Frame>![Plug in ](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/2018-04-10_14-03-44.png)</Frame>



Reboot your server to start the Installation via ISO.

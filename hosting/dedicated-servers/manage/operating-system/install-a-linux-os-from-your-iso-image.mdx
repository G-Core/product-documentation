A custom Linux OS can be installed on a dedicated server by uploading an ISO image to DCImanager and mounting it through the IPMI interface. Windows can only be ordered at server provisioning time and cannot be installed this way.

## Upload the ISO to DCImanager

<Steps>
  <Step title="Open DCImanager">
    In the hosting [Control&nbsp;Panel](https://hosting.gcore.com/billmgr), navigate to **Products/Services** → **Dedicated servers**, select the server, and click **To panel**. DCImanager opens in a new tab.
  </Step>
  <Step title="Navigate to ISO-images">
    In the left sidebar, select **Templates**, then click the **ISO-images** tab.

    <Frame>
      ![DCImanager Templates section with the ISO-images tab open showing the Upload ISO-image button](/images/docs/hosting/dedicated-servers/manage/operating-system/install-a-linux-os-from-your-iso-image/install-a-linux-os-from-your-iso-image-image1.png)
    </Frame>
  </Step>
  <Step title="Upload the ISO">
    Click **Upload ISO-image** and fill in the form:

    - **Name** — enter a name, or leave blank to use the filename from the URL.
    - Select the source:
      - **From the network** — paste the direct download URL of the ISO file.
      - **From the local computer** — select a file from the local machine.
    - **Location** — pre-filled based on the server's datacenter.

    Click **Apply**. The image appears in the ISO-images list when the upload is complete.
  </Step>
</Steps>

## Mount the ISO and boot

<Steps>
  <Step title="Open the IPMI console">
    DCImanager provides access to the server's [IPMI](/hosting/dedicated-servers/manage/log-in-to-ipmi) console. Open it through DCImanager and log in with the IPMI credentials.
  </Step>
  <Step title="Attach the ISO as a virtual drive">
    In the IPMI interface, navigate to the Virtual Media section. Select the option to mount a virtual CD/DVD drive and attach the uploaded ISO file from the server's ISO storage. Confirm the connection.

    <Info>
    The Virtual Media menu location varies by server hardware. On Dell iDRAC, it is under **Configuration** → **Virtual Media**. On Supermicro IPMI, it is under the **Virtual Media** menu in the top bar of the console.
    </Info>
  </Step>
  <Step title="Reboot and boot from the ISO">
    In the IPMI interface, initiate a reboot. Before the server restarts, set the virtual CD/DVD drive as the boot device — most IPMI interfaces offer a **Boot Once** or **Next Boot** option for this. The server will boot from the ISO and start the OS installer.
  </Step>
  <Step title="Complete the installation">
    Follow the installer prompts for the Linux distribution. After the installation finishes, disconnect the virtual media in the IPMI interface and reboot the server to start the newly installed OS.
  </Step>
</Steps>

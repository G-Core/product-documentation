Only users with the Client Administrator or Project Administrator role can invite new users to a project. To send an invitation:

1. In the Gcore Customer Portal, navigate to the [Projects](https://portal.gcore.com/cloud/projects/list) section, click the three-dots menu next to the project, and select **Settings**.


<Frame>![Projects list with the three-dots menu open showing Open and Settings options](/images/docs/cloud/getting-started/projects/users/add-a-new-user-to-your-project/project-settings.png)</Frame>



2. Open the **Access** tab.

3. Click **Invite Users**.


<Frame>![Access tab showing the current users list and the Invite Users button](/images/docs/cloud/getting-started/projects/users/add-a-new-user-to-your-project/invite-users.png)</Frame>



4. Enter the email address, select a role, and click **Send Invite**. To invite multiple users, click **add another user** and repeat for each. Both existing and new users can be invited.


<Frame>![Invite new user dialog with email address and role fields](/images/docs/cloud/getting-started/projects/users/add-a-new-user-to-your-project/add-new-user.png)</Frame>



  * If the user is not registered, Gcore sends an email with a link to create a password. After creating a password, the user can log in.
  * If the user already has a Gcore account, Gcore sends a notification about the newly granted project access. After accepting the invitation, the user can select the project when signing in.

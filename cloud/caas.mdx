Gcore Container as a Service (CaaS) is a cloud service for running containers without Kubernetes or VM management. Gcore handles infrastructure maintenance and scaling automatically.

To use CaaS, choose a supported region, pull the container image, and manage it via the [Gcore Customer Portal](https://portal.gcore.com) or REST API. See [Create a container](/cloud/caas/create-a-container) and [Manage containers](/cloud/caas/manage-containers) for step-by-step instructions.

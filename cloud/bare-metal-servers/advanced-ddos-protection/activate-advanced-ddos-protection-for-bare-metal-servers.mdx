Advanced DDoS Protection filters attack traffic through the Threat Mitigation System before it reaches a [Bare Metal](/cloud/bare-metal-servers/about-bare-metal-servers) server. Protection applies only to public network interfaces or sub-interfaces with public IP addresses — private and internal interfaces do not support this feature.

Before enabling Advanced DDoS Protection during server creation, follow the [creation guide](/cloud/bare-metal-servers/create-a-bare-metal-server). Quota approval is required before the network interface configuration unlocks.

For capabilities, pricing, and how Advanced DDoS Protection differs from basic protection, see the [overview](/cloud/bare-metal-servers/advanced-ddos-protection/about-advanced-ddos-protection-for-bare-metal-servers).

## Enable during server creation

1. In the [Gcore Customer Portal](https://portal.gcore.com), navigate to **Cloud** > **Bare Metal** and click **Create Bare Metal**.

2. Complete the region, image, and server type steps. After quota approval, open the **Configure network interfaces** step.

3. Add a **Public** or **Dedicated public** interface, then enable **Advanced DDoS Protection**.

<Info>
Advanced DDoS Protection is available only on external interfaces and sub-interfaces with public IP addresses. Internal and private interfaces cannot use this feature.
</Info>

4. Open the profile template dropdown and select a template. Each template requires different fields, and the available options depend on the region — pricing details appear when a template is selected.

5. In **Configure access**, select an SSH key for Linux servers or set a password for Windows servers. For key setup, see [SSH keys](/cloud/bare-metal-servers/connect-to-your-bare-metal-server-via-ssh).

6. In **Set quantity and name**, enter the server name template and click **Create Server**.

## Enable on an existing server

1. In the Customer Portal, navigate to **Cloud** > **Bare Metal** and click the server name.

2. Open the **Networking** tab.

3. In the public interface row, open the three-dot menu and select **Edit advanced DDoS protection profile**.

4. In the **Edit advanced DDoS protection profile** dialog, select a profile template from the dropdown, fill in the required fields, and click **Submit**.

The interface row shows **Advanced** as the protection type once activation completes.

## Manage from the DDoS Protection list

1. In the Customer Portal, navigate to **Cloud** > **Networking** > **DDoS Protection**.

2. Locate the Bare Metal server in the list, open the three-dot menu on the right, and select **Edit advanced DDoS protection profile**.

3. In the **Edit advanced DDoS protection profile** dialog, select a profile template and complete the required fields. Profile templates vary by region — for example, Luxembourg-3 offers **Universal v3**, **Universal v5**, and **Universal v6** templates.

4. Click **Submit**. The protection type changes from **Basic** to **Advanced**, and the profile name and parameters appear in the list.

# Create a bare metal server

This guide explains how to create a bare metal server.

## Network requirements

When configuring network for your bare metal server:

- You can create up to 4 bonds per bare metal server
- Each bond supports multiple network interfaces
- Private networks are recommended for internal communication

## Storage options

Bare metal servers support various storage options.

Gcore Cloud uses specific IP addresses for outbound (egress) traffic. If your systems restrict incoming connections, add these addresses to your allowlist.

## When to allowlist Cloud egress IPs

Add Cloud egress IP addresses to your allowlist for:

- **HTTP webhooks** - Cloud sends notifications to your endpoints
- **AMQP connections** - Cloud connects to your message brokers
- **Image downloads** - Cloud fetches images from your servers

## Get the IP address list

Fetch the current list of Cloud egress IP addresses:

```sh
curl -i -X GET https://api.gcore.com/cloud/public/v1/ipranges/egress
```

<Info>
Gcore expands the network continuously, so the list of IP addresses changes. Write a script that automatically updates your allowlist to prevent service disruptions when new addresses are added.
</Info>

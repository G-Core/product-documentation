SSH keys generated via PuTTYgen or in the Gcore Customer Portal must be converted to `.ppk` format before using PuTTY to connect to a Gcore Virtual Machine.

To convert the key, use PuTTYgen:

1\. Make sure [PuTTY](https://putty.org) is installed.

2\. Launch the PuTTYgen app. 

3\. Click **Load** and select the key file to convert.


<Frame>
  ![PuTTYgen app with the Load button highlighted](/images/docs/cloud/virtual-instances/connect/connect-to-your-instance-via-ssh/puttygen-load-key.png)
</Frame>

<Info>
By default, Windows displays only `.ppk` files. To view all file types, click **All files** in the Open file dialog.
</Info>

4\. When prompted to confirm the import, click **OK**.


<Frame>
  ![PuTTYgen confirmation that the key has been imported](/images/docs/cloud/virtual-instances/connect/connect-to-your-instance-via-ssh/puttygen-convert-key.png)
</Frame>



5\. Select a converted key by clicking either **Save private key** or **Save public key**. 


<Frame>
  ![PuTTYgen app with the Save public key and Save private key buttons highlighted](/images/docs/cloud/virtual-instances/connect/connect-to-your-instance-via-ssh/puttygen-save-keys.png)
</Frame>



6\. Save the key for future use.

Gcore Cloud is a suite of cloud computing services that includes Infrastructure as a Service (IaaS) and Platform as a Service (PaaS) environments. Resources are available on demand over the internet with a pay-as-you-go payment model — the underlying hardware is fully managed by Gcore.

The platform covers:

- **Compute** — Virtual Instances and Bare Metal servers
- **Storage** — block volumes, images, File Shares, and Secrets Manager
- **Networking** — load balancers, floating IPs, reserved IPs, firewalls, and private networks
- **Containers and orchestration** — Managed Kubernetes, Container Registry, Container as a Service, and Function as a Service
- **Databases** — Managed PostgreSQL
- **Observability** — Managed Logging

## Benefits

Gcore Cloud is suited for workloads that require fast provisioning, variable capacity, or global distribution:

- **Agility** — resources deploy in minutes across a broad range of technologies.
- **Cost optimization** — payment is based on actual resource usage, converting capital expenses into variable costs.
- **World presence** — resources can be deployed across more than 30 regions worldwide.
- **Elasticity** — resource capacity scales up or down at any time based on workload demand.

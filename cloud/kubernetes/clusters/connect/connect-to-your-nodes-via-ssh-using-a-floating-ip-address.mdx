Kubernetes cluster nodes can be accessed via SSH using [a floating IP address](/cloud/networking/ip-address/create-and-configure-a-floating-ip-address).

## Assign a floating IP address

1\. In the Gcore Customer Portal, navigate to **Cloud** > **Managed Kubernetes**, then click the cluster name.

<Frame>![Screenshot_2023-02-27_at_13.37_2.png](/images/docs/cloud/kubernetes/clusters/connect/connect-to-your-nodes-via-ssh-using-a-floating-ip-address/13321601561489.png)</Frame>

2\. Open the **Pools** section, click the pool to expand it, find the target node, and click **Add IP**.

<Frame>![Screenshot_2023-02-27_at_14.23_1.jpg](/images/docs/cloud/kubernetes/clusters/connect/connect-to-your-nodes-via-ssh-using-a-floating-ip-address/13321623319825.png)</Frame>

3\. In the dialog, create a new floating IP or attach an existing one:

- To create a new floating IP: in the **Attach a floating IP** section, enable the **Attach to existing instance** toggle. Select the node and the interface, then click **Create a new floating IP**.

<Frame>![Screenshot_2023-02-06_at_15.45_2.png](/images/docs/cloud/kubernetes/clusters/connect/connect-to-your-nodes-via-ssh-using-a-floating-ip-address/13321686078225.png)</Frame>

- To attach an existing floating IP: find the required address in the list below the **Attach a floating IP** section and click **Attach to instance**.

<Frame>![Screenshot_2023-02-06_at_15.45_1.png](/images/docs/cloud/kubernetes/clusters/connect/connect-to-your-nodes-via-ssh-using-a-floating-ip-address/13321799776529.png)</Frame>

In the dialog, select the node and the interface, then click **Attach floating IP**.

The floating IP is now attached to the node.

## Connect to a node via SSH

Run the following command, replacing `[floating IP address]` with the node's floating IP and `[path to the private SSH key]` with the path to the private key:

```sh
ssh ubuntu@[floating IP address] -i "[path to the private SSH key]"
```

1\. In the Cloud menu, go to **Kubernetes**, find the required cluster and click its name.


<Frame>![Screenshot_2023-02-27_at_13.37_2.png](/images/docs/cloud/kubernetes/clusters/autoscaling/change-autoscaling-limits/13348527052433.png)</Frame>



2\. Go to the **Pools** section, click â‹¯ next to the pool and select **Change autoscaling limits**. 


<Frame>![Screenshot_2023-02-27_at_20.42_1-2.jpg](/images/docs/cloud/kubernetes/clusters/autoscaling/change-autoscaling-limits/13348601064977.png)</Frame>



3\. In the new window, for **Minimum** and **Maximum number of nodes,** enter the number of nodes you want to maintain and click **Save**.

  * If you have specified more nodes than are in the cluster now, the system will deploy new ones. 
  * If you have specified fewer nodes, the system will remove redundant ones.

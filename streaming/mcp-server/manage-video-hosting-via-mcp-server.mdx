Gcore MCP Server provides video-on-demand management through natural language. AI assistants can upload videos, organize libraries, create playlists, and generate subtitles.

<Info>
Before starting, [install and configure Gcore MCP Server](/account-settings/integrations/install-and-configure-gcore-mcp-server) with `GCORE_TOOLS="streaming.*"` to load streaming tools.
</Info>

## How it works

When a request is made in natural language, the AI client calls the appropriate MCP tool and formats the JSON response for readability.

Example request:

```
List my videos
```

The MCP server returns structured data:

```json
{
  "id": 13755849,
  "name": "Free DEMO: CHARGE - Blender Open Movie",
  "client_id": 1000503,
  "duration": 262730,
  "slug": "cC4PEC0AWy1qu03R",
  "status": "ready",
  "origin_width": 3840,
  "origin_height": 1608,
  "hls_url": "https://1000503.gvideo.io/videos/2675_cC4PEC0AWy1qu03R/master.m3u8",
  "dash_url": "https://1000503.gvideo.io/videos/2675_cC4PEC0AWy1qu03R/master.mpd",
  "converted_videos": [
    {"name": "vod1080n", "width": 2580, "height": 1080, "status": "complete"},
    {"name": "vod720n", "width": 1720, "height": 720, "status": "complete"},

    {"name": "vod360n", "width": 860, "height": 360, "status": "complete"}
  ]
}
```

The AI client presents this as readable text:

## Upload videos

```
Upload video from https://example.com/training/module1.mp4
```

The MCP server initiates the upload and returns status:

```json
{
  "id": 13756001,
  "name": "module1",
  "status": "pending",
  "origin_size": 0,
  "created_at": "2026-04-20T10:15:30.000Z"
}
```

The AI client presents this as:

```
Video upload started

Source: https://example.com/training/module1.mp4
ID: 13756001
Name: module1 (from filename)
Status: downloading

Processing pipeline:
  1. Download: in progress
  2. Transcode: pending
  3. Ready: pending

Estimated time: 10-15 minutes depending on file size.
```

Check progress:

```
What's the status of video 13756001?
```

The server returns current processing state:

```json
{
  "id": 13756001,
  "name": "module1",
  "status": "ready",
  "duration": 184630,
  "origin_width": 1920,
  "origin_height": 1080,
  "hls_url": "https://1000503.gvideo.io/videos/2675_abc123/master.m3u8",
  "converted_videos": [
    {"name": "vod1080n", "status": "complete", "progress": 100},
    {"name": "vod720n", "status": "complete", "progress": 100},
    {"name": "vod480n", "status": "complete", "progress": 100}
  ]
}
```

For upload options, see [Upload videos](/streaming/video-hosting/upload-a-video-and-embed-it-to-your-app).

## Troubleshooting

```
My video has been processing for 2 hours
```

Response:

```
Checking video status...

Video: Conference Recording
Size: 8.5 GB
Duration: 3 hours 20 minutes
Format: 4K H.265
Status: transcoding 1080p (45%)

This is expected for large 4K files:
  - 4K = 4x more pixels than 1080p
  - H.265 requires more processing
  - 3+ hour duration

Estimated completion: 1-2 more hours

For faster processing next time:
  - Upload in 1080p if 4K not needed
  - Use H.264 codec
  - Split long recordings into parts
```

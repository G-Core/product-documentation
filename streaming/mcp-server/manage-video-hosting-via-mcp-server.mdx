Gcore MCP Server provides video-on-demand tools that AI clients can invoke through natural language. The server returns structured data that the AI client interprets and presents in readable format.

<Info>
Before starting, [install and configure Gcore MCP Server](/account-settings/integrations/install-and-configure-gcore-mcp-server) with `GCORE_TOOLS="streaming.*"` to load streaming tools.
</Info>

## Available tools

| Tool | Description |
|------|-------------|
| `streaming_videos_ls` | List all videos |
| `streaming_videos_get` | Get video details by ID |
| `streaming_videos_new` | Upload video from URL |
| `streaming_videos_upd` | Update video metadata |
| `streaming_videos_del` | Delete a video |
| `streaming_playlists_ls` | List playlists |
| `streaming_playlists_new` | Create a playlist |
| `streaming_playlists_upd` | Update playlist content |
| `streaming_playlists_del` | Delete a playlist |
| `streaming_directories_ls` | List directories |
| `streaming_directories_new` | Create a directory |
| `streaming_ai_subtitles_new` | Generate AI subtitles |
| `streaming_ai_translate_new` | Translate subtitles |
| `streaming_statistics_get` | Get viewing statistics |

## View videos

List all videos:

```
List my videos
```

Get details for a specific video:

```
Show details for video 56789
```

The response includes: video ID, name, duration, status (processing/ready), available qualities, view count, and embed URLs.

## Upload videos

Upload from URL:

```
Upload video from https://example.com/video.mp4
```

Upload with metadata:

```
Upload video from https://example.com/demo.mp4 named "Product Demo" with tags demo, product
```

Check processing status:

```
Show processing status for video 56789
```

For upload options, see [Upload videos](/streaming/video-hosting/upload-a-video-and-embed-it-to-your-app).

## Organize library

Create directories:

```
Create directory named "Tutorials"
```

Move videos:

```
Move video 56789 to Tutorials directory
```

Search videos:

```
Find videos with tag "tutorial"
```

## Playlists

Create a playlist:

```
Create playlist "Training Series" with videos 56789, 56790, 56791
```

Add video to playlist:

```
Add video 56792 to playlist "Training Series"
```

For playlist features, see [Playlists](/streaming/video-hosting/create-and-configure-playlists-for-videos).

## AI subtitles

Generate subtitles:

```
Generate subtitles for video 56789
```

Translate subtitles:

```
Translate subtitles for video 56789 to Spanish
```

For subtitle options, see [AI subtitles](/streaming/video-hosting/ai-for-video/generate-ai-subtitles-and-add-them-to-video).

## View statistics

Get viewing metrics:

```
Show statistics for video 56789 for the last 30 days
```

Available metrics include view count, unique viewers, average watch time, completion rate, and geographic distribution.

## Common requests

| Request | Tool called |
|---------|-------------|
| "List my videos" | `streaming_videos_ls` |
| "Upload video from URL" | `streaming_videos_new` |
| "Create playlist with videos" | `streaming_playlists_new` |
| "Generate subtitles" | `streaming_ai_subtitles_new` |
| "Show video statistics" | `streaming_statistics_get` |
| "Delete video 123" | `streaming_videos_del` |

## Troubleshooting

If a request fails, the AI client displays the error message. Common issues:

- **Video not found**: Verify the video ID exists
- **Still processing**: Video upload not yet complete
- **Feature unavailable**: AI subtitles require feature activation

For operations not available through MCP, use the [Gcore Customer Portal](https://streaming.gcore.com) or [Streaming API](/api-reference/streaming).

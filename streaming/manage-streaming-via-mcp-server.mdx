Gcore MCP Server enables AI assistants to manage video streaming resources through natural language commands. This guide covers Streaming-specific tools and example conversations.

For general setup instructions, see [Gcore MCP Server overview](/account-settings/integrations/gcore-mcp-server-overview).

## Configure for Streaming

Set the `GCORE_TOOLS` environment variable to expose Streaming tools:

```bash
GCORE_TOOLS="streaming.*"
```

This exposes approximately 60 Streaming tools covering live streams, VOD, players, and broadcasts.

## Available Streaming tools

| Resource | Operations | Description |
|----------|------------|-------------|
| `streams` | list, create, get, update, delete | Live streams |
| `videos` | list, create, get, update, delete | Video on demand (VOD) |
| `broadcasts` | list, create, get, update, delete | Scheduled broadcasts |
| `players` | list, create, get, update, delete | Video player configurations |
| `playlists` | list, create, get, update, delete | Video playlists |
| `directories` | list, create, get, delete | Content organization |
| `quality_sets` | list, create, get | Encoding quality presets |
| `restreams` | list, create, get, delete | Stream restreaming |
| `statistics` | get | Viewing statistics |

## Example conversations

### List live streams

**Prompt:**
```
List all my live streams
```

**Response:**
```
Found 3 live streams:

1. Main Event Stream (ID: 12345)
   - Status: active
   - URL: rtmp://push.gcore.com/live/stream_12345
   - Viewers: 1,250

2. Backup Stream (ID: 12346)
   - Status: idle
   - URL: rtmp://push.gcore.com/live/stream_12346

3. Test Stream (ID: 12347)
   - Status: idle
```

### Create a live stream

**Prompt:**
```
Create a new live stream named "Product Launch" with HD quality
```

**Response:**
```
Created live stream:
- Name: Product Launch
- ID: 12348
- Status: idle
- Push URL: rtmp://push.gcore.com/live/stream_12348
- Stream Key: abc123xyz
- Playback URL: https://player.gcore.com/stream_12348

Use the push URL and stream key in your streaming software (OBS, etc.).
```

### Manage VOD content

**Prompt:**
```
List all uploaded videos
```

**Prompt:**
```
Upload a video from URL https://example.com/video.mp4 named "Tutorial Part 1"
```

```
Video upload initiated:
- Name: Tutorial Part 1
- ID: 56789
- Status: processing
- Source: https://example.com/video.mp4

Processing typically takes 5-15 minutes depending on video length.

```
Check the processing status of video 56789

## Tool reference

### streaming.streams.list

Lists all live streams.

**Parameters:** None required

**Returns:** Array of streams with ID, name, status, and URLs

### streaming.streams.create

Creates a new live stream.

**Parameters:**
- `name` (required): Stream name
- `quality`: Quality preset (SD, HD, FHD)

### streaming.videos.create

Creates a VOD entry from a URL.

**Parameters:**
- `name` (required): Video name
- `url` (required): Source video URL

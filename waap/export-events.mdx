To export Events, click the download icon in the top-right corner of the page.

<Frame>
  ![WAAP Events export](/images/docs/waap/analytics/waap-events-csv.png)
</Frame>

The currently filtered table data is downloaded in CSV format.

## What data is exported

The exported file contains the events displayed in the table. 

If you apply filters, only the filtered dataset is exported.

## Use cases for exporting Events

Exporting Events data in CSV format is useful to perform custom analysis, pivot reporting, or trend comparisons across time ranges.

### Incident investigation

Export filtered events related to:

- A specific IP address  
- A targeted endpoint  
- A detected attack wave  

Analyze the data offline or share it with a security team for deeper investigation.

### Compliance and audit reporting

Generate a record of:

- Blocked or monitored requests  
- Applied protection actions  
- Attack frequency within a defined time range  

Store exported files as part of internal audit documentation or compliance processes.

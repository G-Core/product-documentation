Gcore MCP Server enables AI assistants to manage FastEdge applications through natural language commands. This guide covers FastEdge-specific tools and example conversations.

For general setup instructions, see [Gcore MCP Server overview](/account-settings/integrations/gcore-mcp-server-overview).

## Configure for FastEdge

Set the `GCORE_TOOLS` environment variable to expose FastEdge tools:

```bash
GCORE_TOOLS="fastedge.*"
```

This exposes approximately 20 FastEdge tools covering edge applications, deployments, and secrets.

## Available FastEdge tools

| Resource | Operations | Description |
|----------|------------|-------------|
| `apps` | list, create, get, update, delete | Edge applications |
| `binaries` | list, upload, get, delete | WebAssembly binaries |
| `kv_stores` | list, create, get, delete | Key-value stores |
| `secrets` | list, create, get, delete | Application secrets |
| `templates` | list, get | Application templates |
| `statistics` | get | Usage statistics |

- Status: deploying

## Tool reference

### fastedge.apps.list

Lists all FastEdge applications.

**Parameters:** None required

**Returns:** Array of applications with ID, name, status, and URL

### fastedge.apps.create

Creates a new FastEdge application.

**Parameters:**
- `name` (required): Application name

### fastedge.secrets.create

Creates a secret for an application.

**Parameters:**
- `app_id` (required): Application ID
- `name` (required): Secret name
- `value` (required): Secret value

### fastedge.kv_stores.create

Creates a key-value store.

**Parameters:**
- `name` (required): KV store name
- `app_id` (required): Application ID

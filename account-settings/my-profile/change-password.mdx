To change the account password:

1. In the Customer Portal, click the avatar in the top-right corner and select **Profile**.

<Frame>![Avatar dropdown menu with Profile option](/images/docs/account-settings/my-profile/change-password/dashboard-account-dialog.png)</Frame>

2. In the **Password** section, click **Change password**.

<Frame>![My profile page with the Password section highlighted](/images/docs/account-settings/my-profile/change-password/change-password-page.png)</Frame>

3. Enter the current password in **Verify current password**, then enter the new password in **New password** and **Confirm new password**.

<Frame>![Change password dialog with current and new password fields](/images/docs/account-settings/my-profile/change-password/change-password-dialog.png)</Frame>

4. Click **Save**.

An email notification confirming the password change will be sent to the account's email address.

<Frame>![Email confirmation of a password change](/images/docs/account-settings/my-profile/change-password/password-change-confirmation.png)</Frame>

## Reset a password

To reset a forgotten password, go to the [reset password page](https://auth.gcore.com/login/forgot-password), enter an email address, and click **Reset password**. A link to create a new password will be sent to that address.

<Frame>![Password reset page with email field and Reset password button](/images/docs/account-settings/my-profile/change-password/password-reset-form.png)</Frame>

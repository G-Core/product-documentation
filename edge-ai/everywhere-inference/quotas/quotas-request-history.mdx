The **Quotas Request History** page shows all quota increase requests submitted for your account. You can use it to track the status of pending requests and review past changes.

## Open Quotas Request History

In the Gcore Customer Portal, navigate to **Everywhere Inference** > **Quotas** > **Quotas Request History**.

<Frame>![Quotas Request History page with no requests submitted yet](/images/docs/edge-ai/everywhere-inference/quotas/quotas-request-history/quotas-request-history-1.png)</Frame>

## Request list

Each row in the request list shows:

- **Date** — when the request was submitted.
- **Resources** — the quota resources included in the request (for example, Inference GPU L40S Count, Inference Instances Count).
- **Requested amount** — the quota value requested for each resource.
- **Status** — the current state of the request.

Possible statuses:

| Status | Description |
|---|---|
| Pending | The request has been submitted and is awaiting review. |
| Approved | The quota increase has been applied to your account. |
| Rejected | The request was not approved. Contact Gcore support for details. |

## Create a new request

To submit a new quota increase request, go to **Quotas** > **[Quotas Viewer](/edge-ai/everywhere-inference/quotas/request-quota-increase)**, select the resources you need, and fill in the request form.

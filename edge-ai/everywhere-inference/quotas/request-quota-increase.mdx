New Gcore accounts start with a zero quota for Everywhere Inference. You must request a quota increase before you can create deployments.

The **Quotas** section has two pages:

- **Quotas Viewer** — view your current resource limits and submit quota increase requests.
- **[Quotas Request History](/edge-ai/everywhere-inference/quotas/quotas-request-history)** — track the status of submitted requests.

## Step 1. Open the Quotas Viewer

In the Gcore Customer Portal, navigate to **Everywhere Inference** > **Quotas** > **Quotas Viewer**.

<Frame>![Account Quotas page showing available resources and the Request form](/images/docs/edge-ai/everywhere-inference/quotas/request-quota-increase/request-quota-increase-1.png)</Frame>

The page shows all available quota resources and their current usage in the format **used / limit**:

- **Inference CPU Millicore Count** — total vCPU millicores allocated across all deployments.
- **Inference GPU A100 Count** — number of A100 GPUs.
- **Inference GPU H100 Count** — number of H100 GPUs.
- **Inference GPU L40S Count** — number of L40S GPUs.
- **Inference Instances Count** — total number of deployments.

## Step 2. Select the resources to increase

Click **+** next to the resource you want to increase. The resource appears in the **Selected resources** panel on the right.

<Frame>![Account Quotas page with L40S GPU and Instances Count selected in the right panel](/images/docs/edge-ai/everywhere-inference/quotas/request-quota-increase/request-quota-increase-2.png)</Frame>

Use the **+** and **−** buttons in the **Selected resources** panel to set the requested quota value for each resource.

### Quota calculation example

For a deployment with the following settings:

- Flavor: 1xL40S / 16 vCPU / 232 GiB RAM
- Regions: 2
- Pods per region: 1

The deployment runs two pods (one per region), so you need at least:

- 1 deployment (Inference Instances Count)
- 20000 millicores (Inference CPU Millicore Count)
- 2 L40S GPUs (Inference GPU L40S Count)

If autoscaling is set to a maximum of 2 pods per region, request:

- 1 deployment
- 40000 millicores
- 4 L40S GPUs

<Info>
**Info**

If you are unsure about your requirements, request more than you think you need so you can adjust autoscaling settings later.
</Info>

## Step 3. Submit the request

In the **Request** panel, enter a description explaining your use case, then click **Send request**.

Gcore processes quota requests in up to 15 minutes. After the quota is updated, you can create deployments or modify autoscaling settings for existing ones.

To track the status of your request, go to **Quotas** > **[Quotas Request History](/edge-ai/everywhere-inference/quotas/quotas-request-history)**.

# Gclaw overview

Gclaw is a managed [OpenClaw](https://github.com/openclaw/openclaw) service with built-in inference, powered by Gcore infrastructure. It enables you to launch AI agents instantly without manual configuration, SSH access, or VPS setup.

<Info>
Gclaw is currently in beta. During the beta period, each account is limited to one Gclaw instance, and service levels, guarantees, or support commitments do not apply.
</Info>

## Key features

Gclaw provides the following capabilities:

- **One-click launch**: Get a fully configured OpenClaw instance in seconds. Click **Start** and begin chatting within approximately one minute.

- **Built-in inference**: Inference is already connected and running using Kimi K2.5 and other AI models. You do not need to find a provider, manage API tokens, or optimize usage.

- **Dedicated environment**: Each user gets an isolated virtual machine with a personal OpenClaw instance. Your data and configuration are fully separated from others.

- **Messaging integrations**: Connect your AI agent to Telegram, Discord, WhatsApp, and other messaging platforms to interact with it where it is convenient.

- **Configuration management**: Back up and restore your configuration, restart your instance, or recreate it from scratch using the Settings panel.

## Use cases

Gclaw is suitable for:

- Personal AI assistants that can be accessed from multiple messaging platforms
- Prototyping AI agent workflows without infrastructure management
- Testing OpenClaw features with built-in inference
- Automating tasks through conversational interfaces

## What's included

Gclaw includes the following third-party products:

- [OpenClaw](https://github.com/openclaw/openclaw) - An open-source AI agent framework
- [Kimi K2.5](https://github.com/MoonshotAI/Kimi-K2.5) - A large language model for inference

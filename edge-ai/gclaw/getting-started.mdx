# Getting started with Gclaw

This guide walks you through launching your first Gclaw instance and getting started with your personal AI assistant.

## Prerequisites

Before you begin, ensure you have:

- A Gcore account. If you do not have one, [sign up for free](https://auth.gcore.com/login/signup).

## Launch your instance

To create and launch your Gclaw instance:

1. In the Gcore Customer Portal, navigate to **AI** > **Gclaw**.

2. If this is your first time accessing Gclaw, review the Terms of Use and click **I Agree**.

3. Click **Get started for free**.

4. Wait approximately one minute for your instance to be prepared. You will see the message "Your assistant is being prepared. This usually takes a few moments..."

5. Once ready, you will see a welcome message from Gclaw and the chat interface will become active.

<Info>
During the beta period, no credit card is required to use Gclaw.
</Info>

## Explore the interface

After your instance is ready, you will see the Gclaw chat interface with the following elements:

- **Chat area**: Displays the conversation between you and your AI assistant. The welcome message provides tips for getting started.

- **Message input**: Type your messages and questions here. Press Enter or click Send to submit.

- **Settings button**: Click the gear icon to access instance management options.

## Get started with your assistant

Your Gclaw assistant can help you with various tasks. Here are some things you can try:

### Back up before making changes

Before adding new skills or connections, create a backup of your configuration:

1. Click the **Settings** button (gear icon).
2. Click **Backup Config**.

If something breaks, you can restore your previous configuration.

### Set up messaging integrations

Ask your assistant for configuration snippets to connect messaging platforms:

- "How do I connect Telegram?"
- "How do I connect Discord?"
- "How do I connect WhatsApp?"

Your assistant will provide step-by-step instructions with copy-paste configuration commands.

### Enable web search

To enable your assistant to search the web, you need to add a Brave Search API key:

1. Get a Brave Search API key from [Brave Search API](https://brave.com/search/api/).
2. Ask your assistant: "Help me enable web search."
3. Follow the provided instructions to add your API key.

## About memory

Gclaw starts fresh each session and does not remember previous conversations by default. Your assistant will keep notes in the workspace to build continuity over time.

To preserve important information between sessions, consider:

- Using the backup and restore features
- Asking your assistant to take notes
- Configuring persistent storage options

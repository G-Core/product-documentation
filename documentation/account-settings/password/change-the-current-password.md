# Change or reset a password

To change the password for your account: 

1\. In the <a href="https://accounts.gcore.com/reports/dashboard" target="_blank">Gcore Customer Portal</a>, navigate to the account settings in the top-right corner of the screen, and click **Profile**.

<img src="https://assets.gcore.pro/docs/account-settings/password/change-the-current-password/dashboard-account-menu.png" alt="Profile settings in the Customer Portal" width="80%">

2\. Open the **Change password** page.

<img src="https://assets.gcore.pro/docs/account-settings/password/change-the-current-password/change-password-page.png" alt="Change password page" width="80%">

3\. Enter the current password once and the new password twice.

4\. Click **Save** to apply the updates.

The password has been changed. You will receive an email notification informing you about the password change.


<img src="https://assets.gcore.pro/docs/account-settings/password/change-the-current-password/password-change-confirmation.png" alt="Email confirmation of a password change" width="60%">

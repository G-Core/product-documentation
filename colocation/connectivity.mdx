Gcore offers connectivity services for colocation customers who need internet access or private networking between sites. Each service is billed separately from rack space and power.

## IP Transit

IP Transit provides access to the global internet through a high-capacity connection between the customer network and the Gcore backbone. Traffic can reach any destination worldwide using Gcore's IP transit and peering links.

IP Transit requires the customer to have their own Autonomous System Number (ASN) and network equipment such as a router.

Available port speeds: 1G, 10G, 40G, 100G, 400G. Billing is based on a committed data rate (minimum 10% of port capacity), with bursts measured using the 95th percentile method. Port usage is charged as a one-time fee, with the committed bandwidth billed monthly.

## IP Access

IP Access provides internet connectivity for customers who do not have their own ASN. Gcore assigns the IP addresses and handles routing on the customer's behalf.

It's suited for customers with a single server or a small deployment who need straightforward internet access without managing their own network equipment or BGP configuration. IP Access is commonly ordered together with colocation.

Available port speeds and billing follow the same model as IP Transit.

## Cross-connects

A cross-connect is a dedicated physical cable connection between two parties within the same data center — for example, between customer equipment and a carrier, an internet exchange, or another tenant.

Cross-connects are provisioned using fiber or copper and billed as a one-time installation fee plus a monthly recurring charge for the active connection.

## Leased lines

A leased line is a dedicated, private point-to-point connection between two locations. It provides guaranteed bandwidth and consistent performance, unlike shared internet connections.

Leased lines are billed monthly based on bandwidth, distance, and service type, with an optional installation fee.

## Order connectivity services

Contact the [sales team](https://gcore.com/contact-sales) with the following details to request any connectivity service:

- Location (data center)
- Required bandwidth and port speed
- Service type (IP Transit, IP Access, cross-connect, or leased line)
- Number of links (two links to separate routers are recommended for redundancy)
- For IP Transit: ASN and BGP details

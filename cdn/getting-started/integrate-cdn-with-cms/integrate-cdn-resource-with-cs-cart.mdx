---
title: Integrate CDN resource with Cs-cart
sidebarTitle: Cs-cart
ai-navigation: Install the Gcore CDN Universal Add-on plugin for Cs-cart, configure the CDN URL field with a CNAME record from Gcore Customer Portal, and enable the plugin to rewrite static file URLs through the CDN resource.
---

Before you take any steps please back up your files and database. The plugin works only with default CMS pattern. If you manually changed CMS patterns plugin might not help you.

You need to buy the plugin (unfortunately, there are no free plugins at the moment). You can buy it [here](https://marketplace.cs-cart.com/add-ons/site-management/content-delivery-network-cs-cart-addon.html). Install the plugin.  
Login to the CMS administration panel and go the Add-ons sections, Manage Add-ons. 


<Frame>![CMS administration panel ](/images/docs/cdn/getting-started/integrate-cdn-with-cms/integrate-cdn-resource-with-cs-cart/integrate-cdn-resource-with-cs-cart-image2.png)</Frame>



Click on **+**.


<Frame>![Plus Button](/images/docs/cdn/getting-started/integrate-cdn-with-cms/integrate-cdn-resource-with-cs-cart/integrate-cdn-resource-with-cs-cart-image3.png)</Frame>



Choose the way to install the file. Click **Upload & Install**. 


<Frame>![Upload & Install](/images/docs/cdn/getting-started/integrate-cdn-with-cms/integrate-cdn-resource-with-cs-cart/integrate-cdn-resource-with-cs-cart-image4.png)</Frame>



Having installed Universal CDN Add-on, go to its settings. 


<Frame>![settings](/images/docs/cdn/getting-started/integrate-cdn-with-cms/integrate-cdn-resource-with-cs-cart/integrate-cdn-resource-with-cs-cart-image5.png)</Frame>



In the CDN URL field insert CNAME that you specified in the [Gcore Customer Portal](https://portal.gcore.com/accounts/reports/dashboard).

Ensure that your [CNAME record has been configured](/cdn/cdn-resource-options/general/create-and-set-a-custom-domain-for-the-content-delivery-via-cdn) in a proper way before using it for integration.


<Frame>![Apply settings](/images/docs/cdn/getting-started/integrate-cdn-with-cms/integrate-cdn-resource-with-cs-cart/integrate-cdn-resource-with-cs-cart-image1.png)</Frame>



Apply settings and enable the plugin.

Integration has been completed! We highly recommend you to check the HTML code of your web page to ensure that URLs have been rewritten properly from your original ones to CNAME from the Gcore Customer Portal.

To do that press **F12** or open Developers Tools in your browser, choose the Network tab and refresh the page. All static files should have your CNAME in URLs.

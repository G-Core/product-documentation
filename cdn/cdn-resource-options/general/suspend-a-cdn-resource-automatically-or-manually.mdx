## Automatic CDN resource suspension

If a resource is active, but there has not been traffic in 90 days, it will be automatically suspended by the system.

All users with administrator and engineer roles will receive an email 7 days before resource suspension, listing the resources scheduled for suspension.

In the resource settings you will see the time when it will be stopped:

<Frame>![ resource settings ](/images/docs/cdn/cdn-resource-options/general/suspend-a-cdn-resource-automatically-or-manually/cdn-resource-suspended.png)</Frame>


After the resource suspension, all users with the administrator and engineer roles will receive an email with a list of stopped resources.

<Note>
**Note** 

If you start to deliver traffic via the resource after receiving the first email:

  * The second email about the suspension will not be sent.
  * The resource will not be suspended.
</Note>


After enabling a resource that was suspended, you will see a message that the CDN resource was suspended because there has not been traffic in 90 days.

If there is still no traffic after enabling the resource, it will be suspended again in 7 days.

To avoid suspension of the resources without traffic, contact [Gcore Support](https://support.gcore.com/hc/en-us/requests/new).

## Manual CDN resource suspension

You can suspend a CDN Resource from [Gcore Customer Portal](https://portal.gcore.com/). 

Open the CDN **Resources settings** for the resource you want to suspend. When **Content availability** is enabled, you will see that the content is available to customers. 


<Frame>![CDN Resources settings Content availability](/images/docs/cdn/cdn-resource-options/general/suspend-a-cdn-resource-automatically-or-manually/cdn-content-availability-enabled.png)</Frame>


Click the **Content availability** toggle, then select **Yes, suspend**.


<Frame>![disable your CDN Resource](/images/docs/cdn/cdn-resource-options/general/suspend-a-cdn-resource-automatically-or-manually/cdn-resource-yes-suspend.png)</Frame>

It will display that the content is not available to customers.

<Frame>![CDN Resources settings Content availability](/images/docs/cdn/cdn-resource-options/general/suspend-a-cdn-resource-automatically-or-manually/cdn-content-availability-disabled.png)</Frame>


Click **Save changes**.


<Frame>![disable your CDN Resource](/images/docs/cdn/cdn-resource-options/general/suspend-a-cdn-resource-automatically-or-manually/cdn-save-changes.png)</Frame>


The settings will be applied within 30-60 seconds.

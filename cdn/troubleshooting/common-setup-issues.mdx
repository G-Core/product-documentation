## Common Setup Issues

### 503 Service Unavailable
- Verify origin server is accessible directly
- Check origin allows CDN IP ranges
- Confirm correct origin port (80 or 443)

### ERR_TOO_MANY_REDIRECTS
- Both CDN and origin redirecting HTTP->HTTPS
- Solution: Disable redirect on one side

### Website downloads as file
- Origin returning wrong Content-Type
- Check origin server MIME types

### SSL Certificate errors
- DNS propagation takes 24-48 hours
- Let's Encrypt needs DNS to validate domain
- Wait and retry

## What is Edge Proxy

Edge Proxy is a DDoS protection solution that runs on the Gcore CDN edge infrastructure. It allows you to protect your applications by routing traffic through a globally distributed proxy layer.

You can enable protection in minutes without deploying your own infrastructure or configuring complex networking.

To start using Edge Proxy, follow the instructions in the [Setup and configuration](/edge-proxy/setup-and-configuration) guide.

## How Edge Proxy works

Edge Proxy uses the CDN edge network to receive and filter incoming traffic before forwarding it to your origin server.

Key principles:

- Traffic is routed through Anycast IP addresses to the nearest Point of Presence (PoP)
- Filtering and proxying happen at the edge
- Clean traffic is forwarded to your origin server
- You receive a protected IP address and port to expose to users

## Key benefits

### High availability and scalability

Edge Proxy runs on the CDN infrastructure, which distributes traffic across multiple PoPs. This removes regional bottlenecks and enables multi-terabit filtering capacity.

### Lower latency

Traffic is processed closer to end users. This reduces round-trip time and improves application responsiveness.

### Simple setup

You can configure protection in a few steps without managing virtual machines or networking components.

### Efficient IP usage

Edge Proxy uses shared Anycast IP pools, reducing the number of dedicated IP addresses required.

### Flexible configuration

You can create multiple translation rules for a single protected IP address.

## When to use Edge Proxy

Edge Proxy is suitable for:

- Game servers
- APIs and backend services
- Small and medium-sized applications
- Services that require fast DDoS protection without complex setup

## Core concepts

### Protected IP

A public IP address with a port assigned by Gcore. Users connect to this endpoint.

### Origin server

Your backend server that receives filtered traffic.

### Translation rule

A configuration that maps:

- Origin IP
- Port
- Protocol
- Application type

One combination equals one rule.

### Application type

A predefined countermeasure profile optimized for specific applications.

## What you get after setup

After enabling Edge Proxy, you receive:

- A protected IP address and port
- Active DDoS protection
- A configuration summary
- The ability to manage and delete proxies

Gcore bills S3 Standard, SFTP, and S3 Fast as separate products — each appears as an independent line item on the monthly invoice. Charges are based on the amount of data stored, outgoing traffic, and API requests.

## Pricing

S3 Standard and SFTP use GB for billing, while S3 Fast uses GiB (1 GiB = 1.074 GB).

### S3 Standard

General-purpose storage for most workloads. Three plans are available: PAYG, Startup+, and Professional+.

PAYG charges based on actual usage with no monthly commitment:

| Item | Price (EUR) | Price (USD) |
|------|-------------|-------------|
| Storage | €0.04/GB/month | $0.044/GB/month |
| Egress | €0.02/GB | $0.022/GB |
| Egress to CDN | Free | Free |
| Ingress | Free | Free |
| Requests | €0.03/10K | $0.033/10K |

Startup+ (€120/month) includes 5,000 GiB of S3 Standard storage; usage above the quota is charged at the overage rate:

| Item | Price (EUR) |
|------|-------------|
| Storage overage | €0.024/GB |
| Egress | €0.02/GB |
| Egress to CDN | Free |
| Ingress | Free |
| Requests | €0.03/10K |

Professional+ (€400/month) includes 20,000 GiB of S3 Standard storage and is designed for higher-volume workloads:

| Item | Price (EUR) |
|------|-------------|
| Storage overage | €0.02/GB |
| Egress | €0.02/GB |
| Egress to CDN | Free |
| Ingress | Free |
| Requests | €0.03/10K |

Startup+ and Professional+ quotas cover S3 Standard storage only; SFTP usage is always billed separately.

### SFTP

SFTP is available on PAYG only — there are no commit plans for SFTP. SFTP usage is not included in the Startup+ or Professional+ quota.

SFTP uses the same PAYG rates as S3 Standard: €0.04/GB/month for storage, €0.02/GB for egress, free ingress, and €0.03/10K for requests.

### S3 Fast

High-performance storage optimized for AI/ML workloads, IoT telemetry, and latency-sensitive applications, available in Sines-2, Sines-3, and London-2.

| Item | Price (EUR) | Price (USD) |
|------|-------------|-------------|
| Storage | €0.064/GiB/month | $0.07/GiB/month |
| Egress | €0.0013/GB | $0.00143/GB |
| Ingress | Free | Free |
| Requests | €0.03/10K | $0.033/10K |

<Note>
Self-commit plans are not currently available for S3 Fast storage. For committed volume discounts, contact the [sales&nbsp;team](https://gcore.com/contact-sales).
</Note>

## Billing and plan management

Storage consumption is metered continuously throughout the month — S3 Standard and SFTP in GB-hours, S3 Fast in GiB-hours. Usage charges and subscription fees are applied on the 2nd of the following month. If payment fails, the service is suspended.

### Plan changes

To change a storage plan, contact support via chat or email at [support@gcore.com](mailto:support@gcore.com). The new plan takes effect on the 1st day of the following month.

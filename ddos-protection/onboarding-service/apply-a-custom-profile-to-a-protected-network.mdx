---

## Overview

After creating a custom protection profile, apply it to a protected network.

## Instructions

### Step 1. Open protected networks

1. In the [Gcore Customer Portal](https://accounts.gcore.com/reports/dashboard), navigate to **DDoS Protection** \> **Protected networks**.
2. Locate the protected network to update.

### Step 2. Edit the protected network

1. Next to the target network, click **â€¦**.
2. Select **Edit**.

<Frame>
  ![Edit the protected network](/images/docs/ddos-protection/protected-networks-edit.png)
</Frame>

### Step 3. Assign a protection profile

1. In the **Edit protected network** view, open the **Profile** list.
2. Select the required protection profile.

The list includes:

- Basic profile (default)
- Custom profiles

### Step 4. Save changes

1. Review the selected profile.
2. Click **Save changes**.

The selected profile becomes active immediately.

<Frame>
  ![Save the protected network](/images/docs/ddos-protection/protected-network-save.png)
</Frame>

## Verification

- Confirm that the network status reflects the updated profile.
- Monitor traffic to verify that the profile works as expected.
- Adjust the profile settings if required.

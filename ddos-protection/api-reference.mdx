## List protection profiles

Retrieve a list of all DDoS protection profiles for your account.

### HTTP Request

```http
GET /ddos/profiles
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `managed_by` | string | No | Filter profiles by who manages them. Valid values: `system`, `user` |

#### managed_by values

- **`system`**: System-managed protection profiles (default profiles provided by Gcore)
- **`user`**: User-created custom protection profiles

### Response

The API returns a list of protection profiles with their configuration details.

```json
{
  "profiles": [
    {
      "id": "profile-123",
      "name": "Default Web Profile",
      "managed_by": "system",
      "description": "System-managed profile for web applications"
    },
    {
      "id": "profile-456", 
      "name": "Custom Gaming Profile",
      "managed_by": "user",
      "description": "User-created profile optimized for gaming traffic"
    }
  ]
}
```

### Examples

#### Get all profiles

```bash
curl -X GET "https://api.gcore.com/ddos/profiles" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

#### Get only system-managed profiles

```bash
curl -X GET "https://api.gcore.com/ddos/profiles?managed_by=system" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

#### Get only user-created profiles

```bash
curl -X GET "https://api.gcore.com/ddos/profiles?managed_by=user" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

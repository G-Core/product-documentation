# Connect Telegram to Gclaw

Telegram integration allows interaction with the Gclaw AI assistant directly from Telegram. This requires creating a Telegram bot and configuring the Gclaw instance with the bot token.

<Tip>
**Tip**

Ask the Gclaw assistant directly: "How do I connect Telegram?" The assistant provides interactive instructions in the chat.
</Tip>

## Prerequisites

This procedure requires a running Gclaw instance (see [Getting started](/gclaw/getting-started)) and a Telegram account.

## Step 1: Create a Telegram bot

To create a bot and obtain a bot token:

1. Open Telegram and search for **@BotFather**.

2. Start a chat with BotFather and send the command `/newbot`.

3. Follow the prompts to configure the bot:
   - Enter a display name (such as "My Gclaw Assistant")
   - Enter a username ending with "bot" (such as "my_gclaw_bot")

4. After successful creation, BotFather sends a message containing the bot token. The token looks like: `123456789:ABCdefGHIjklMNOpqrSTUvwxYZ`

5. Copy and save this token for the next step.

<Warning>
**Warning**

Keep the bot token secure. Anyone with access to this token can control the bot.
</Warning>

## Step 2: Add the token to OpenClaw

Gclaw is a managed service without direct SSH access to the underlying VM. To configure the Telegram bot token, ask the Gclaw assistant in the chat interface:

"Help me connect Telegram with my bot token"

The assistant provides the appropriate configuration commands and applies them to the instance.

<Frame>
<img src="/images/docs/gclaw/gclaw-telegram-instructions.png" alt="Gclaw providing Telegram connection instructions" />
</Frame>

<Info>
**Info**

If the assistant is unable to apply the configuration, contact [Gcore support](mailto:support@gcore.com) with the bot token for manual configuration.
</Info>

## Step 3: Connect a Telegram account

Gclaw uses pairing mode by default to connect Telegram accounts:

1. Open Telegram and find the newly created bot by searching for its username.

2. Send any message to the bot (such as "Hello").

3. The bot responds with a pairing code.

4. Follow the assistant's instructions to approve the pairing.

## Verify the connection

After completing the pairing:

1. Send a test message to the bot in Telegram.

2. The Gclaw assistant should respond to the message.

<Warning>
**Warning**

If no response is received, verify that the bot token is correct, the pairing was completed successfully, and the Gclaw instance is running. If the instance is unresponsive, click **Restart Instance** in the Settings panel.
</Warning>

## Advanced configuration

Additional Telegram configuration options are available by asking the Gclaw assistant:

- "How do I set up group support for Telegram?"
- "Configure my Telegram bot to require @mentions in groups"
- "How do I allow multiple accounts on my Telegram bot?"

The assistant provides specific configuration commands for each option.

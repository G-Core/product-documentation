# Getting started with Gclaw

Gclaw provides a fully managed OpenClaw instance with built-in inference. The instance is ready to use within approximately one minute after clicking **Start**.

## Prerequisites

A Gcore account is required. To create an account, [sign up for free](https://auth.gcore.com/login/signup).

## Launch an instance

To create and launch a Gclaw instance:

1. In the Gcore Customer Portal, navigate to **AI** > **Gclaw**.

2. If this is the first time accessing Gclaw, review the Terms of Use and click **I Agree**.

3. Click **Get started for free**.

4. Wait approximately one minute for the instance to be prepared.

5. Once ready, a welcome message appears and the chat interface becomes active.

## Interface elements

The chat area displays the conversation with the AI assistant, including a welcome message with tips for getting started. The message input field at the bottom of the screen accepts text—press Enter or click the Send button to submit messages.

The gear icon in the top-right corner opens the Settings panel, which provides access to instance management options including Restart Instance, Backup Config, Restore Config, and Recreate Instance.

## Start chatting

The welcome message includes several tips for working with Gclaw. Click the gear icon to access Backup Config and Restore Config—create a backup before making significant changes to enable rollback if something breaks.

Ask the assistant for copy-paste config snippets to connect messaging platforms like Telegram, Discord, or WhatsApp. To enable web search capabilities, add a [Brave Search API key](https://brave.com/search/api/) by asking the assistant "Help me enable web search."

Gclaw starts fresh each session and does not remember previous conversations by default. The assistant keeps notes in the workspace to build continuity over time.

## Ask for help

Questions can be asked directly in the chat. The assistant provides step-by-step instructions with copy-paste commands for tasks like connecting Telegram, connecting Discord, enabling web search, or backing up configuration.

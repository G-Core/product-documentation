# Getting started with Gclaw

Gclaw provides a fully managed OpenClaw instance with built-in inference. The instance is ready to use within approximately one minute after clicking **Start**.

## Prerequisites

A Gcore account is required. To create an account, [sign up for free](https://auth.gcore.com/login/signup).

## Launch an instance

To create and launch a Gclaw instance:

1. In the Gcore Customer Portal, navigate to **AI** > **Gclaw**.

2. If this is the first time accessing Gclaw, review the Terms of Use and click **I Agree**.

3. Click **Get started for free**.

4. Wait approximately one minute for the instance to be prepared.

5. Once ready, a welcome message appears and the chat interface becomes active.

## Interface elements

The Gclaw chat interface contains the following elements:

- **Chat area**: Displays the conversation with the AI assistant. The welcome message provides tips for getting started.

- **Message input**: Located at the bottom of the screen. Press Enter or click the Send button to submit messages.

- **Settings button**: The gear icon in the top-right corner provides access to instance management options including Restart Instance, Backup Config, Restore Config, and Recreate Instance.

## Start chatting

The Gclaw assistant can help with various tasks. Key capabilities from the welcome message:

- **Settings and Backups**: Click the gear icon to find Backup Config and Restore Config buttons. Back up before making significant changes to enable rollback if something breaks.

- **Setting Up Connections**: Ask the assistant for copy-paste config snippets to connect messaging platforms like Telegram, Discord, or WhatsApp.

- **Enable Web Search**: Add a [Brave Search API key](https://brave.com/search/api/) to enable web search capabilities. Ask the assistant "Help me enable web search" for instructions.

- **About Memory**: Gclaw starts fresh each session and does not remember previous conversations by default. The assistant keeps notes in the workspace to build continuity over time.

## Ask for help

Questions can be asked directly in the chat:

- "How do I connect Telegram?"
- "How do I connect Discord?"
- "Help me enable web search"
- "How do I back up my configuration?"

The assistant provides step-by-step instructions with copy-paste commands.

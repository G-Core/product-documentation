Some Gclaw skills require external API keys to function. The Gclaw assistant can add API keys directly to the OpenClaw configuration.

## Add an API key

Provide the key in the chat interface:

> "Add my OpenAI key: sk-abc123..."

The assistant stores the key securely in the instance configuration.

<Frame>
<img src="/images/docs/gclaw/api-keys-config.png" alt="Gclaw showing API key configuration options" />
</Frame>

## Supported integrations

| Integration | Configuration |
|-------------|---------------|
| OpenAI (DALL-E, GPT, Whisper) | API key |
| Spotify | Client ID, Client Secret, Redirect URI |
| Brave Search | API key for web search |
| GitHub | Personal access token |
| Notion | Integration token |

Ask the assistant about specific configuration requirements for other integrations.
